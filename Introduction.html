Turning a Computer On and Off Properly

The very first skill — and one of the most important

Before learning any other computer skill, you must first know how to properly start and shut down a computer. Many beginners damage computers by switching them off incorrectly. Learning the correct procedure protects the system and your files, and it also makes you look confident and professional at a cybercafé or workplace.

🖥️

Desktop Computer

Has separate parts: a monitor (screen), a system unit (CPU box), a keyboard, and a mouse. Usually found in offices, schools, and cybercafés.

💻

Laptop Computer

A portable computer with everything built into one device — the screen, keyboard, touchpad (mouse), and battery are all together. You can carry it anywhere.

A Turning On a Desktop Computer

Follow these steps carefully, in this exact order:

Step-by-Step: Starting a Desktop
1

Ensure the computer is connected to a power source (wall socket or extension cord).

2

Switch on the wall socket or power extension.

3

Press the power button on the System Unit (the large box, usually on the front).

4

Press the power button on the Monitor (the screen, usually at the bottom or side).

5

Wait for the computer to load the Operating System (e.g., Windows). This may take 30–60 seconds.

6

If a password screen appears, type your password and press Enter. The desktop will appear when the computer is ready.

B Turning On a Laptop

A laptop is simpler because everything is in one unit:

Step-by-Step: Starting a Laptop
1

Ensure the battery is charged, or plug in the charger cable before starting.

2

Open the laptop lid (if it is closed).

3

Press the power button (usually at the top-right of the keyboard area or on the side).

4

Wait for the system to boot up (start). Enter your password if required.

Tip: Check the Battery First

Always check that a laptop has enough battery charge before starting. A laptop that dies mid-task can corrupt your files. Aim to keep the battery above 20%.

C Shutting Down a Computer Properly — Very Important!

Never switch off the power directly from the socket while the computer is still running. This is one of the most common mistakes beginners make — and it can damage your files and the computer itself.

Correct Shutdown Procedure (Windows)
1

Save any open files before you begin to shut down.

2

Click the Start Button at the bottom-left corner of the screen (looks like a Windows logo).

3

Click the Power icon (a circle with a line at the top).

4

Select Shut Down from the menu.

5

Wait until the screen goes completely black — do not touch anything during this time.

6

(Desktop only) Switch off the monitor, then switch off the wall socket.

Why Proper Shutdown Matters
  • Prevents file corruption and data loss
  • Allows Windows to save settings properly
  • Protects the hard drive from damage
  • Keeps the computer healthy for longer
Common Mistakes to Avoid
  • Switching off from the wall while the computer is running
  • Holding the power button to force shutdown (only as a last resort)
  • Closing the laptop lid without shutting down first
  • Pulling out the power cable of a laptop without saving work
Other Power Options You Should Know
🔄 Restart

Shuts down and starts again. Use when a program freezes or after installing updates.

💤 Sleep

Saves your work to memory and uses very little power. Wakes up quickly. Good for short breaks.

🏔️ Hibernate

Saves everything to the hard drive and turns off completely. Takes longer to wake up. Good for overnight.

✏️

Practice Activity 1 — Power On & Off

Do this with a real computer. Repeat until you are confident.

1

Turn on a computer correctly following the steps above.

2

Log in to the system (enter password if required).

3

Shut down the computer properly — without assistance — using the Start → Power → Shut Down method.

4

Try it a second time, this time using keyboard shortcut: press Win + X, then choose Shut Down.

I can start a laptop correctly

I understand the difference between Shut Down, Restart, Sleep, and Hibernate

Using the Mouse and Keyboard Correctly

Your two main tools for communicating with a computer

The mouse and keyboard are the primary input devices used to communicate with a computer. Mastering them improves your speed and confidence enormously. Most people only learn to use one or two fingers — this chapter will help you use them properly.

A Using the Mouse

The mouse controls the pointer (the small arrow) on the screen. When you move the mouse on the desk, the pointer moves on the screen in the same direction.

🖱️

Standard Mouse

Left button — main click
Scroll wheel — scroll up/down
Right button — options menu
👆
PointingMove the mouse to position the pointer over an item on screen — without clicking.
🖱️
Clicking (Left Click)Press the LEFT mouse button once. Used to select items, click buttons, and follow links.
⚡
Double-ClickingPress the LEFT button twice quickly. Used to open files, folders, and programs.
📋
Right-ClickingPress the RIGHT button once. Opens a menu with extra options (like copy, paste, rename).
✋
DraggingHold the LEFT button while moving the mouse. Used to move files and resize windows.
🔄
ScrollingRoll the scroll wheel up or down to move up or down a page — without moving the pointer.
Holding the Mouse Correctly

Place your hand gently on the mouse — don't grip it tightly. Your index finger rests on the left button, your middle finger on the scroll wheel or right button. Keep your wrist relaxed on the desk.

🖱️

B Using the Keyboard

The keyboard is used to type text, numbers, and commands. A standard keyboard has about 104 keys, divided into sections. Learning where each key is and what it does will make you much faster.

Main Sections of the Keyboard
🔤
Typing Keys

Letters A–Z and numbers 0–9. The main area used for writing documents and emails.

⚙️
Function Keys (F1–F12)

The top row. Each has a special purpose depending on the program you are using.

🧭
Navigation Keys

Arrow keys (↑↓←→), Home, End, Page Up, Page Down. Used to move around a document.

🔢
Numeric Keypad

On the right side of full-size keyboards. Used for entering numbers quickly, like a calculator.

Typing Tips for Beginners
✓Keep your wrists straight and relaxed on the desk — do not bend them up or down.
✓Try to use all fingers, not just your index fingers. It will feel slow at first but gets much faster.
✓Look at the screen, not at your hands, as much as possible.
✓Practice for 10–15 minutes every day. Consistency beats long sessions.
✓Use a typing practice sentence: "The quick brown fox jumps over the lazy dog" — it contains every letter of the alphabet!

Understanding the Desktop Environment

Your digital workspace — everything starts here

After turning on the computer and logging in, the first screen you see is called the Desktop. Think of the desktop as your digital office table — it is where you arrange your work, open programs, and access files. Everything you do on a computer starts from here.

Icons

Small pictures that represent programs, files, or folders. Examples: This PC, Recycle Bin, Microsoft Word. Double-click to open. Right-click for options.

📌

Taskbar

The bar at the bottom of the screen. Contains the Start Button, pinned programs, open programs, the clock, and notification area. Used to switch between open programs.

🪟

Start Button

Found at the bottom-left corner. Click it to open programs, search for files, access Settings, and shut down the computer. Press the Win key as a shortcut.

🖼️

Desktop Background (Wallpaper)

The image or colour behind the icons. You can right-click → Personalize to change the wallpaper to a photo or colour you prefer.

🔔

Notification Area (System Tray)

At the bottom-right of the taskbar. Shows the time, date, Wi-Fi status, battery level (on laptops), and notification icons from programs running in the background.

🔍

Search Bar

Next to the Start Button. Click and type to search for programs, files, settings, or even websites — it is the fastest way to find anything on your computer.

B Creating and Managing Folders on the Desktop

Folders help you organise your files — just like physical folders in a filing cabinet. Always put related files in the same folder so you can find them easily later.

Create a New Folder
1

Right-click on an empty space on the desktop (not on an icon).

2

Move the mouse to New in the menu.

3

Click Folder from the submenu.

4

A new folder appears. Type a name for it (e.g., "My Work").

5

Press Enter to confirm.

Rename a Folder
1

Right-click on the folder you want to rename.

2

Click Rename from the menu.

3

The name becomes highlighted — type the new name.

4

Press Enter to save the new name.

Quick rename shortcut:

Click once to select a folder, then press F2 to rename it immediately.

Common Desktop Mistakes to Avoid
  • Accidentally deleting important icons (right-click → Undo or check the Recycle Bin to restore)
  • Saving ALL files directly on the desktop without organising them into folders
  • Double-clicking too slowly — it may open a rename instead of opening the file. Try again slightly faster.
  • Having too many icons on the desktop — it slows down the computer and makes things hard to find

Opening and Closing Programs

Programs are the tools you use to get work done

Programs (also called applications or apps) are software used to perform specific tasks — like typing documents, browsing the internet, listening to music, or creating spreadsheets. Every task you do on a computer uses a program.

📄

Microsoft Word

For writing documents, letters, and CVs

📊

Microsoft Excel

For spreadsheets, calculations, and budgets

📽️

PowerPoint

For creating presentations and slide shows

🌐

Google Chrome

For browsing the internet and websites

📝

Notepad

Simple text editor — great for quick notes

📁

File Explorer

For browsing and managing files and folders

A Three Ways to Open a Program
Method 1: Start Menu
1

Click the Start Button.

2

Scroll through the list of installed programs.

3

Click the program you want (e.g., Microsoft Word).

4

Wait for the program window to appear.

Method 2: Search
1

Click the Search bar next to Start (or press Win key).

2

Type the name of the program (e.g., "Excel").

3

Click the program in the search results.

⚡ Fastest method — just type and press Enter!
Method 3: Desktop Shortcut
1

Find the program icon on the desktop.

2

Double-click the icon.

3

The program will open.

💡 Right-click any program in the Start Menu → Pin to desktop to add a shortcut.

When any program opens, you will see a window. Every window has the same controls at the top-right corner — learning these saves you a lot of confusion.

C Closing a Program — Always Save First!

Before closing any program, always save your work first. If you forget, Windows will usually ask "Do you want to save?" — always click Save.

Method 1: Close Button

Click the X at the top-right corner of the window. If prompted, click Save before the window closes.

Method 2: File Menu
1

Click File at the top-left of the program.

2

Click Exit or Close.

Method 3: Keyboard Shortcut

Press Alt + F4 to close the active window immediately. Save first!

Working with Multiple Programs

You can have several programs open at the same time — they will all appear in the Taskbar at the bottom. Click any program in the Taskbar to switch to it. This is called multitasking. For example, you can have Word, Excel, and a browser all open at once.
