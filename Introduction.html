1.1 Turning a Computer On and Off Properly

Introduction
Before learning any computer skill, you must first know how to properly start and shut down a computer. Many beginners damage computers by switching them off incorrectly. Learning the correct procedure protects the system and your files.

A. Turning On a Desktop Computer

A desktop computer has separate parts: monitor, system unit (CPU box), keyboard, and mouse. Follow these steps:

  1. Ensure the computer is connected to a power source.
  2. Switch on the wall socket or power extension.
  3. Press the power button on the System Unit (CPU box).
  4. Press the power button on the Monitor.
  5. Wait for the computer to load the Operating System (e.g., Windows).
  6. If required, enter your password and press Enter. The desktop screen will appear when the computer is ready for use.

B. Turning On a Laptop

A laptop is portable and has everything built into one device. Steps:

  1. Ensure the laptop battery is charged or connect it to power.
  2. Press the power button.
  3. Wait for the system to boot.
  4. Enter your password if required.

C. Shutting Down a Computer Properly (Very Important)

Never switch off the power directly from the socket unless the computer is completely off.

Correct Shutdown Procedure (Windows):

  1. Click the Start Button (bottom-left corner).
  2. Click Power.
  3. Select Shut Down.
  4. Wait until the screen goes completely black.
  5. Switch off the monitor (desktop users).
  6. Switch off the wall socket.
Why Proper Shutdown Is Important
Prevents file corruption, hardware damage, and data loss. Always shut down properly.
Common Mistakes to Avoid
- Switching off from the wall socket while the computer is still running
- Forcing shutdown by holding the power button (only as last resort)
- Closing laptop lid without proper shutdown (may cause issues)

1.2 Using the Mouse and Keyboard Correctly

Introduction
The mouse and keyboard are the primary input devices used to communicate with a computer. Mastering them improves speed and confidence.

A. Using the Mouse

The mouse controls the pointer on the screen.

Main Mouse Actions

  • Pointing - Moving the mouse to position the pointer.
  • Clicking - Pressing the left button once.
  • Double-clicking - Pressing the left button twice quickly.
  • Right-clicking - Pressing the right mouse button once.
  • Dragging - Holding the left button while moving the mouse.
  • Scrolling - Using the scroll wheel to move up or down a page.

Practical Exercise 2

  1. Move the pointer to the Recycle Bin icon.
  2. Double-click to open it.
  3. Close the window by clicking the X button.
  4. Right-click on the desktop.
  5. Click anywhere else to remove the menu.

B. Using the Keyboard

Main Sections of the Keyboard

  • Typing keys (letters A-Z, numbers 0-9)
  • Function keys (F1-F12)
  • Navigation keys (Arrow keys, Home, End, Page Up, Page Down)
  • Numeric keypad (on the right side)

Important Keyboard Keys and Their Uses

  • Enter - Confirms a command or moves to a new line
  • Spacebar - Inserts a space between words
  • Backspace - Deletes characters to the left of the cursor
  • Delete - Deletes characters to the right of the cursor
  • Shift - Types capital letters or symbols above numbers
  • Caps Lock - Locks capital letters on
  • Tab - Moves cursor to the next tab stop or field
  • Ctrl - Used with other keys for shortcuts (e.g., Ctrl+C for Copy)
  • Alt - Used with other keys for menu shortcuts
  • Windows Key - Opens the Start Menu

Basic Typing Practice
Type the following sentences without looking at your hands (if possible):
"The quick brown fox jumps over the lazy dog."
"My name is __________. I am learning to type."

Beginner Typing Tips

  • Keep your wrists straight and relaxed
  • Use all fingers, not just your index fingers
  • Look at the screen, not at your hands
  • Practice regularly for 10-15 minutes daily

1.3 Understanding the Desktop Environment

Introduction
After turning on the computer and logging in, the first screen you see is called the Desktop. The desktop is your main working area. It contains tools that help you open programs, manage files, and control the computer. Think of the desktop as a digital office table where you arrange your work.

A. Main Parts of the Desktop

I. Icons
Icons are small pictures that represent programs, files, or folders.
Examples: This PC, Recycle Bin, Microsoft Word, Web Browser
To open an icon, double-click it.

II. Taskbar
The taskbar is located at the bottom of the screen.
It contains: The Start Button, Open programs, Time and date, Notification area.
The taskbar helps you switch between open programs.

III. Start Button
The Start Button is found at the bottom-left corner of the screen.
It allows you to: Open programs, Access settings, Shut down the computer, Search for files.

IV. Desktop Background
This is the image or color behind the icons. It can be changed according to user preference.

Practical Activity 4: Exploring the Desktop
Follow these steps:

  1. Identify the Start Button.
  2. Identify at least three icons on the desktop.
  3. Click the time and date on the taskbar.
  4. Open and close one desktop icon.

Repeat until comfortable.

Creating a New Folder on the Desktop

Folders help you organize files.

Steps:

  1. Right-click on an empty space on the desktop.
  2. Select New.
  3. Click Folder.
  4. Type your name as the folder name.
  5. Press Enter.

You have now created a folder successfully.

Renaming a Folder

  1. Right-click the folder.
  2. Select Rename.
  3. Type a new name.
  4. Press Enter.
Common Mistakes to Avoid
- Deleting important icons accidentally
- Saving all files directly on the desktop without organizing them
- Double-clicking too slowly (it may not open)

1.4 Opening and Closing Programs

Introduction
Programs (also called applications) are software used to perform specific tasks such as typing documents, browsing the internet, or creating spreadsheets.

Examples: Microsoft Word, Microsoft Excel, Microsoft PowerPoint, Google Chrome

A. Opening a Program Using the Start Menu

Follow these steps:

  1. Click the Start Button.
  2. Scroll through the list of programs.
  3. Click the program you want to open (e.g., Microsoft Word).
  4. Wait for the program window to appear.

B. Opening a Program Using Search

  1. Click the Start Button.
  2. Type the name of the program (e.g., Word).
  3. Click the program from the search results.

C. Opening a Program from a Desktop Shortcut

  1. Locate the program icon on the desktop.
  2. Double-click the icon.
  3. The program will open.

D. Understanding the Program Window

When a program opens, you will see:

  • Title Bar - Displays the name of the program
  • Minimize Button - Sends the window to the taskbar
  • Maximize Button - Expands the window to fill the screen
  • Close Button (X) - Closes the program

E. Closing a Program Properly

Always save your work before closing.

Method 1: Using the Close Button
Click the X at the top-right corner. If prompted, click Save before exiting.

Method 2: Using File Menu
Click File. Select Exit.

Method 3: Using Keyboard Shortcut
Press Alt + F4 on the keyboard. (This closes the active window.)
