1.1 Turning a Computer On and Off Properly

Introduction: Before learning any computer skill, you must first know how to properly start and shut down a computer. Many beginners damage computers by switching them off incorrectly. Learning the correct procedure protects the system and your files.

A. Turning On a Desktop Computer
  • Ensure the computer is connected to a power source.
  • Switch on the wall socket or power extension.
  • Press the power button on the System Unit (CPU box).
  • Press the power button on the Monitor.
  • Wait for the computer to load the Operating System (e.g., Windows).
  • Enter your password if required.
B. Turning On a Laptop
  • Ensure the laptop battery is charged or connect it to power.
  • Press the power button.
  • Wait for the system to boot.
  • Enter your password if required.
C. Shutting Down Properly

Correct Shutdown Procedure (Windows):

  • Click the Start Button → Power → Shut Down.
  • Wait until the screen goes completely black.
  • Switch off the monitor (desktop users).
  • Switch off the wall socket.

1.2 Using the Mouse and Keyboard Correctly

Introduction: The mouse and keyboard are the primary input devices used to communicate with a computer. Mastering them improves speed and confidence.

A. Using the Mouse
  • Pointing – Move the pointer.
  • Clicking – Left button once.
  • Double-clicking – Left button twice quickly.
  • Right-clicking – Right button once.
  • Dragging – Hold left button while moving.
  • Scrolling – Use scroll wheel.

B. Using the Keyboard
  • Typing keys (letters, numbers)
  • Function keys (F1–F12)
  • Navigation keys (arrows, Home, End)
  • Numeric keypad

Important Keys: Enter, Spacebar, Backspace, Delete, Shift, Caps Lock, Tab, Ctrl, Alt, Windows Key.

1.4 Opening and Closing Programs

Introduction: Programs (applications) are software used to perform specific tasks such as typing documents, browsing the internet, or creating spreadsheets.

Examples: Microsoft Word, Microsoft Excel, Microsoft PowerPoint, Google Chrome

A. Opening a Program Using the Start Menu
  • Click the Start Button.
  • Scroll through the list of programs.
  • Click the program you want to open.
B. Opening a Program Using Search
  • Click the Start Button.
  • Type the name of the program.
  • Click the program from the search results.
C. Opening a Program from a Desktop Shortcut
  • Locate the program icon on the desktop.
  • Double-click the icon.
D. Understanding the Program Window
  • Title Bar: Displays the name of the program.
  • Minimize: Sends the window to the taskbar.
  • Maximize: Expands the window to fill the screen.
  • Close (X): Closes the program.
E. Closing a Program Properly
  • Method 1: Click the X at the top-right corner.
  • Method 2: File → Exit.
  • Method 3: Keyboard shortcut Alt + F4.
