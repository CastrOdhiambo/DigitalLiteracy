When using a computer, you create many documents — letters, reports, spreadsheets, pictures, and presentations. These are called files. To avoid confusion and loss of work, files must be properly organised. Computers use folders to store and arrange files neatly — just like a physical filing cabinet.

Good file management helps you find your work quickly, avoid losing important documents, keep your computer tidy, and work more professionally.

Section 2.1

Understanding Files and Folders

What Is a File?

A file is a saved piece of work stored on a computer. Every file you create or download is stored as a file on the computer's hard drive or external storage.

Examples of common files:

  • A Word document (e.g., your CV or an assignment)
  • An Excel spreadsheet (e.g., a budget or grade sheet)
  • A PowerPoint presentation (e.g., a school project)
  • A picture or photo (.jpg or .png)
  • A PDF document (e.g., a form or certificate)

Every file has three parts:

File Anatomy
  • ① File Name — The title you give the file (e.g., My_CV)
  • ② File Extension — Shows the file type (e.g., .docx means Word)
  • ③ Storage Location — Where it is saved (e.g., Desktop, Documents)
Example of a File Name

My_CV.docx
My_CV is the file name  ·  .docx is the file extension (tells Windows this is a Word document)

Common File Extensions You Must Know

File extensions tell Windows (and you) which program opens that file. Understanding extensions helps you know exactly what type of document you are dealing with.

What Is a Folder?

A folder is a container used to store files on a computer. Folders help you group related documents together so they are easy to find. Folders can also contain other folders — these are called subfolders.

📁 Example: A Well-Organised Folder Structure
📁 My Documents
  ├── 📁 School Work
    ├── 📄 English_Essay.docx
    ├── 📊 Maths_Assignment.xlsx
    └── 📑 Science_Report.pdf
  ├── 📁 Work
    ├── 📄 My_CV.docx
    └── 📊 Budget_2026.xlsx
  └── 📁 Personal
    └── 🖼️ ID_Photo.jpg

Opening File Explorer

File Explorer is the Windows tool for viewing and managing all your files and folders. Think of it as the digital equivalent of opening a filing cabinet drawer.

Method 1 — Start Menu
  1. Click the Start Button
  2. Look for File Explorer in the list
  3. Click it to open
Method 2 — Keyboard Shortcut

Hold the Windows Key and press E:

⊞ Win+E

Fastest way — memorise this!

Parts of the File Explorer Window

  • Navigation Pane (left side) — Shows folders, drives, and This PC
  • Main Window (centre) — Displays the contents of the selected folder
  • Address Bar (top) — Shows your current location (e.g., This PC > Documents)
  • Search Bar (top right) — Type a file name to find it instantly
1 Practical Activity 1 — Explore File Explorer
  1. Open File Explorer using Windows Key + E
  2. Click on Documents in the left navigation pane
  3. Identify at least three files stored there
  4. Look at the Address Bar — what location does it show?
  5. Close File Explorer when done

Section 2.2

Creating Files and Folders

Creating a New Folder

Before saving any work, you must know how to create a proper storage location. Creating folders is one of the most important organisational skills on a computer.

Method 1 — On the Desktop
  1. Go to the Desktop
  2. Right-click on an empty space
  3. Select New
  4. Click Folder
  5. Type the folder name (e.g., Digital Skills)
  6. Press Enter
Method 2 — Inside Documents
  1. Open File Explorer
  2. Click Documents
  3. Right-click empty space
  4. Select New → Folder
  5. Type a name (e.g., School Work)
  6. Press Enter

Creating a New File (Using Microsoft Word)

To create a file, you must open a program first. Here we use Microsoft Word as an example — the same steps apply for Excel and PowerPoint.

Steps to Create and Save a Word Document
  1. 1 Open Microsoft Word from the Start Menu
  2. 2 Click Blank Document
  3. 3 Type your content (e.g., your name, a short paragraph)
  4. 4 Click the File tab at the top left
  5. 5 Select Save As
  6. 6 Choose the location (e.g., Documents or Desktop)
  7. 7 Type a clear file name (e.g., Practice Document)
  8. 8 Click Save — your file is now created!

Good File Naming Rules

Naming files correctly saves you time and prevents confusion. Follow these rules every time you save a file.

Good Naming Habits
  • Use clear, meaningful names
  • Include the topic and date (e.g., Budget_Jan_2026)
  • Use underscores _ instead of spaces
  • Keep names short but descriptive
  • Use capital letters for readability
Avoid These Mistakes
  • Avoid vague names like Document1
  • Don't use symbols: / \ : * ? " < > |
  • Don't use very long names
  • Never leave files unnamed
  • Avoid saving everything as Untitled
Good vs Bad File Names

✅ Term_One_Report_2026.docx — clear, organised, dated
❌ Document12345.docx — confusing, unprofessional
✅ Castro_CV_Final.docx — easy to identify
❌ asdfgh.docx — you will never find this again!

2 Practical Activity 2 — Create Your First Organised Folder

  2. Inside it, create a subfolder called Computer Studies
  3. Open Microsoft Word and type your full name and today's date
  4. Save the file inside the Computer Studies folder with the name My_First_Document
  5. Open File Explorer and confirm the file is saved in the right location

Section 2.3

Renaming, Copying, Moving & Deleting Files

A. Renaming a File or Folder

Renaming helps you correct mistakes, update file names, or improve organisation. You can rename any file or folder you own.

How to Rename
  1. 1 Locate the file or folder you want to rename
  2. 2 Right-click on it
  3. 3 Select Rename from the menu
  4. 4 The name will become editable — type the new name
  5. 5 Press Enter to confirm
Faster Way to Rename

Click the file once to select it, then press F2 on the keyboard. The name becomes editable immediately!

B. Copying a File

Copying creates a duplicate of the file. The original stays in its location and a copy appears in the new location. This is useful for making backups.

Method 1 — Right-Click
  1. Right-click the file
  2. Select Copy
  3. Open the destination folder
  4. Right-click inside the folder
  5. Select Paste
Method 2 — Keyboard
  1. Click the file once
  2. Press Ctrl+C to Copy
  3. Open the destination folder
  4. Press Ctrl+V to Paste

Fastest and most professional method!

C. Moving a File

Moving transfers the file to a new location. Unlike copying, no duplicate is created — the file disappears from the original location and appears in the new one.

Method 1 — Cut and Paste
  1. Right-click the file
  2. Select Cut
  3. Open the new location
  4. Right-click and select Paste
Method 2 — Drag & Drop
  1. Click and hold the file
  2. Drag it to the new folder
  3. Release the mouse button

Best when both folders are visible on screen.

Copy vs Move — What's the Difference?

Copy (Ctrl+C then Ctrl+V) — original stays, a duplicate appears in the new location.
Cut (Ctrl+X then Ctrl+V) — original is removed and appears in the new location only. No duplicate.

D. Deleting a File

Deleting removes a file from its current location and places it in the Recycle Bin. It is not permanently deleted until you empty the Recycle Bin.

Method 1 — Delete Key
  1. Click the file once to select it
  2. Press Delete on the keyboard
  3. The file goes to the Recycle Bin
Method 2 — Right-Click
  1. Right-click the file
  2. Select Delete from the menu
  3. Confirm if asked

E. Restoring a Deleted File (Recycle Bin)

If you delete a file by mistake, don't panic! Files in the Recycle Bin can be fully restored to their original location.

How to Restore from Recycle Bin
  1. 1 Double-click the Recycle Bin icon on the Desktop
  2. 2 Find the file you want to restore
  3. 3 Right-click the file
  4. 4 Select Restore
  5. 5 The file returns to its original location automatically
Warning — Permanent Deletion

If you Empty the Recycle Bin, all files inside are permanently deleted and cannot be recovered. Only empty the Recycle Bin when you are 100% sure you no longer need those files.

Section 2.4

Using a USB Flash Disk

What Is an External Storage Device?

External storage devices allow you to store, transfer, and back up files outside the computer. The most common one in Kenya — especially at cybercafés — is the USB Flash Disk (also called a thumb drive or flash drive).

Common External Devices
  • 🔌 USB Flash Disk (most common)
  • 💾 External Hard Drive
  • 📱 Memory Card (SD card)
  • ☁️ Cloud Storage (Google Drive, OneDrive)
Why Use a Flash Disk?
  • Save work at a cybercafé to take home
  • Transfer files between computers
  • Back up important documents
  • Share files without using the internet

Inserting and Opening a Flash Disk

Inserting the Flash Disk
  1. 1 Locate a USB port on the computer (usually on the front or side)
  2. 2 Insert the flash disk carefully — it only goes in one way
  3. 3 Wait a few seconds for Windows to detect it
  4. 4 A notification may appear — click it to open the flash disk

Opening the Flash Disk (if no notification appears)

Via File Explorer
  1. 1 Press ⊞ Win+E to open File Explorer
  2. 2 Click This PC in the left panel
  3. 3 Under Devices and Drives, find your flash disk (e.g., USB Drive E:)
  4. 4 Double-click it to open

Copying Files To and From a Flash Disk

Computer → Flash Disk
  1. Locate the file on your computer
  2. Right-click → Copy
  3. Open the flash disk window
  4. Right-click inside → Paste
Flash Disk → Computer
  1. Open the flash disk
  2. Right-click the file → Copy
  3. Open your Documents folder
  4. Right-click → Paste

Safely Removing a Flash Disk — VERY IMPORTANT

Never just pull out a flash disk! Always eject it properly first. Removing it while files are being read or written can corrupt your files or damage the flash disk itself.

Safe Ejection Steps
  1. 1 Look at the taskbar (bottom-right corner of screen)
  2. 2 Click the USB / Eject icon (looks like a USB plug with a checkmark)
  3. 3 Click Eject [your flash disk name]
  4. 4 Wait for the message: "Safe to Remove Hardware"
  5. 5 Now remove the flash disk carefully
Why Safe Removal Matters
  • Prevents file corruption — your documents won't be damaged
  • Protects the flash disk from physical damage
  • Ensures all data has been fully written before removal
  • Avoids losing work you thought was saved
4 Practical Activity 4 — Flash Disk Practice
  1. Insert a flash disk into the computer's USB port
  2. Open it using File Explorer → This PC
  3. Create a new folder inside it called Backup Files
  4. Copy one document from your computer into the Backup Files folder
  5. Confirm the file is inside the flash disk folder
  6. Safely eject the flash disk using the taskbar USB icon

Section 2.5

Searching and Organising Files Efficiently

Searching for a File

As you use a computer more, you will store many files. Being able to find a specific file quickly is a critical digital skill — especially in a professional environment.

Method 1 — File Explorer Search
  1. Open File Explorer
  2. Navigate to the location (e.g., Documents)
  3. Click the Search Bar (top-right)
  4. Type the file name or a keyword
  5. Press Enter — results appear instantly
Method 2 — Start Menu Search
  1. Click the Start Button
  2. Type the file name directly
  3. Click the file from the results

Windows searches your entire computer — fast!

Sorting Files

Sorting arranges files in a useful order so you can find them faster — by name, date, type, or size.

How to Sort Files in a Folder
  1. 1 Open the folder containing your files
  2. 2 Right-click on an empty space inside the folder
  3. 3 Select Sort by
  4. 4 Choose your preferred sorting method (see table below)
Sort ByWhat It DoesBest Used When
NameAlphabetical order A–ZYou know the file name
Date ModifiedMost recently changed firstFinding your latest work
TypeGroups by file extensionSeparating Word, Excel, PDFs
SizeLargest to smallestFreeing up storage space

Professional File Organisation Tips

Following these habits consistently will keep your computer fast, clean, and professional — whether you are a student, job seeker, or business owner.

Best Practices
  • Create folders for each category (School, Work, Personal)
  • Use clear, dated file names
  • Save work immediately — don't wait
  • Back up to a flash disk or Google Drive weekly
  • Delete unnecessary files to free up space
  • Keep the Desktop clean — use it temporarily only
Common Mistakes
  • Saving files without checking the location
  • Using unclear or auto-generated file names
  • Forgetting to back up important work
  • Keeping too many files on the Desktop
  • Never deleting old files — leads to clutter
  • Using one folder for everything
Cybercafé Tip — Always Take Your Work With You

At a cybercafé, computers are shared. Always save your work to a flash disk or email it to yourself before leaving. Never save personal files on a public cybercafé computer — another user may accidentally delete them or access your personal information.

5 Practical Activity 5 — Organise and Search
  1. Create three folders named: School, Work, Personal Projects
  2. Move any existing files into the correct folders
  3. Use the Search Bar in File Explorer to find a specific file by name
  4. Open the School folder and sort files by Date Modified
  5. Delete at least one file you no longer need
  6. Review your Desktop — move any files there into appropriate folders
