How to Tell Text from Numbers

Numbers align to the right by default. Text aligns to the left. If your numbers are left-aligned, Excel sees them as text and they won't work in formulas!

Entering and Navigating Data

Keys After Typing

AutoFill — Excel's Time-Saving Feature

AutoFill lets Excel automatically complete a series for you. Instead of typing 1, 2, 3, 4, 5... you type just 1 and 2, then drag.

Using AutoFill
  1. 1 Type the first two values of your series (e.g., 1 and 2, or Jan and Feb)
  2. 2 Select both cells
  3. 3 Hover over the bottom-right corner of the selection — the cursor becomes a small + (Fill Handle)
  4. 4 Click and drag down or right to fill the series
What AutoFill Can Complete Automatically
  • Numbers: 1, 2, 3… or 5, 10, 15…
  • Months: January, February, March…
  • Days: Monday, Tuesday, Wednesday…
  • Dates: 01/01/2026, 02/01/2026…
  • Formulas: copies and adjusts cell references automatically

Section 4.3

Formulas & Functions — Excel's Core Power

The Golden Rule of Excel Formulas

Every formula MUST start with an equals sign =

Without the =, Excel treats what you type as text, not a formula.
✅ =A1+B1 → calculates the sum    ❌ A1+B1 → displays the text "A1+B1"

Arithmetic Operators

OperatorMeaningExampleResult (if A1=10, B1=5)
+Addition=A1+B115
-Subtraction=A1-B15
*Multiplication=A1*B150
/Division=A1/B12
^Power / Exponent=A1^2100
%Percentage=A1*15%1.5

Understanding Cell Ranges

Colon (:) — Range

A1:A10 means "from A1 to A10" — all 10 cells in between. Used in functions like =SUM(A1:A10)

Comma (,) — Multiple

A1,C1,E1 means "A1 and C1 and E1" — non-adjacent cells. Used like =SUM(A1,C1,E1)

Essential Functions — Learn All of These

=SUM()Math
Adds all numbers in a range. The most-used function in Excel.
=SUM(B2:B10)
→ Total of all values from B2 to B10
=AVERAGE()Statistical
Calculates the mean (average) of a range of numbers.
=AVERAGE(C2:C10)
→ Sum ÷ Count of all values in C2:C10
=MAX()Statistical
Returns the largest number in a range.
=MAX(D2:D20)
→ Highest score / largest value
=MIN()Statistical
Returns the smallest number in a range.
=MIN(D2:D20)
→ Lowest score / smallest value
=COUNT()Statistical
Counts how many cells contain numbers.
=COUNT(B2:B20)
→ Number of students with a score entered
=COUNTA()Statistical
Counts all non-empty cells (text or numbers).
=COUNTA(A2:A20)
→ Total number of entries in column A
=IF()Logical
Tests a condition — if TRUE gives one result, if FALSE gives another.
=IF(B2>=50,"Pass","Fail")
→ "Pass" if score ≥ 50, "Fail" if below 50
=ROUND()Math
Rounds a number to a specified number of decimal places.
=ROUND(AVERAGE(B2:B10),1)
→ Average rounded to 1 decimal place
=CONCATENATE()Text
Joins text from multiple cells. Modern alternative: use & operator.
=A2&" "&B2
→ "Faith Achieng" (joins first + last name)
=TODAY()Date
Returns today's date, automatically updating every time you open the file.
=TODAY()
→ Shows current date (e.g., 05/04/2026)

The Fill Handle — Copy Formulas Instantly

After writing a formula in one cell, you can copy it to all other rows automatically using the Fill Handle. Excel adjusts the cell references automatically (this is called relative referencing).

How to Use the Fill Handle
  1. 1 Write your formula in the first cell (e.g., E2 = =SUM(B2:D2))
  2. 2 Click on cell E2 to select it
  3. 3 Move your mouse to the bottom-right corner of E2 — cursor becomes a +
  4. 4 Click and drag down through E3, E4, E5 etc.
  5. 5 Release — Excel fills the formula for every row, adjusting B3:D3, B4:D4, etc.

Common Formula Errors and What They Mean

ErrorMeaningHow to Fix
#VALUE!Formula contains text where a number is expectedCheck cells in the formula — make sure all contain numbers
#DIV/0!Dividing by zero or an empty cellCheck the divisor cell — it must not be 0 or empty
#REF!Cell reference is invalid (deleted row/column)Update or re-enter the formula with correct references
#NAME?Excel doesn't recognise the function nameCheck spelling — =SUME() is wrong, =SUM() is right
######Column is too narrow to display the numberWiden the column by dragging the column border
#N/AValue not available (common in VLOOKUP)Check the lookup value exists in the source table

Formatting Cells Professionally

Number Formatting — Display Data Correctly

Formatting does not change the actual number stored in a cell — it only changes how it looks. The number 1000 can be displayed as 1,000 or Ksh 1,000.00 or 100000% — it's still the same value inside.

How to Apply Number Formatting
  1. 1 Select the cells you want to format
  2. 2 Go to Home Tab → Number group
  3. 3 Click the Number Format dropdown (shows "General" by default)
  4. 4 Choose your format, OR click "More Number Formats" for advanced options

Font, Borders, and Fill Colour

Font Formatting
  • Bold: Ctrl+B — for headings and totals
  • Italic: Ctrl+I — for notes
  • Font Size: Home Tab → Font Size box
  • Font Colour: Home Tab → Font Color (A▾)
  • Increase/Decrease Size: Ctrl+Shift+> / <
Borders
  • Select cells → Home Tab → Borders dropdown (⊞▾)
  • All Borders — grid lines on all cells
  • Outside Borders — box around the selection
  • Thick Box Border — bold outer border
  • Best practice: All Borders for data, Thick Box for the whole table
Fill Colour (Background)
  • Select cells → Home Tab → Fill Color (bucket▾)
  • Use light colours only — dark fills make text hard to read
  • Headers: light blue or light green
  • Totals row: light yellow or pale green
  • Alternating rows: very light grey
Alignment
  • Horizontal: Left / Centre / Right (Home Tab)
  • Vertical: Top / Middle / Bottom
  • Merge & Centre: combines selected cells into one — great for table titles
  • Wrap Text: shows long text on multiple lines within one cell

Conditional Formatting — Colour Cells Automatically

Conditional formatting automatically applies colour, bold, or other styles to cells based on their value. For example, highlight all failing scores in red — without doing it manually.

Applying Conditional Formatting
  1. 1 Select the cells you want to apply rules to (e.g., the Scores column)
  2. 2 Go to Home Tab → Conditional Formatting
  3. 3 Choose a rule type:
    Highlight Cells Rules → Greater Than, Less Than, Equal To, Between…
    Top/Bottom Rules → Top 10%, Above Average…
    Data Bars / Colour Scales / Icon Sets → visual indicators
  4. 4 Set the condition (e.g., Less Than 50) and choose a colour (e.g., Red fill)
  5. 5 Click OK
Practical Use of Conditional Formatting
  • Red background for scores below 50 (Fail)
  • Green background for scores above 80 (Distinction)
  • Data Bars in the Amount column of a budget to see which category is largest
  • Top 3 values highlighted in gold on a sales leaderboard

Creating a Chart — Step by Step

Insert a Chart
  1. 1 Select your data — include the headers (e.g., A1:B6)
  2. 2 Go to Insert Tab
  3. 3 Click the chart type button (e.g., Insert Column Chart)
  4. 4 Choose a style (e.g., 2-D Clustered Column)
  5. 5 The chart appears on your sheet — click and drag to reposition it

Customising Your Chart

  • Chart Title: Click the title text box → type a meaningful name (e.g., "Monthly Sales — April 2026")
  • Axis Titles: Click chart → Chart Design Tab → Add Chart Element → Axis Titles → Primary Horizontal/Vertical
  • Data Labels: Add Chart Element → Data Labels → show the exact value on each bar/slice
  • Legend: Shows which colour represents which data series — can be repositioned or hidden
  • Chart Style: Chart Design Tab → choose from the style gallery to change colours and appearance
  • Resize: Click the chart → drag corner handles to resize
  • Move to new sheet: Right-click chart → Move Chart → New Sheet (gives the chart its own full-page tab)
Chart Mistakes to Avoid
  • Using a Pie chart with more than 7 categories — it becomes unreadable
  • Not adding a chart title — nobody knows what the chart is about
  • Not labelling axes — readers can't understand the scale
  • Using 3D charts — they look fancy but distort perception of values
  • Selecting the wrong data range — your chart shows nonsense

Section 4.6

Sorting and Filtering Data

Sorting Data

Sorting rearranges your rows in a meaningful order — alphabetically, by highest score, by date, etc. Excel moves entire rows so related data stays together.

Quick Sort (1 Column)
  1. Click any cell in the column you want to sort by
  2. Go to Data Tab
  3. Click Sort A to Z (ascending) or Sort Z to A (descending)

Excel automatically selects all related columns and sorts all rows together.

Multi-Level Sort
  1. Click any cell in your data
  2. Data Tab → Sort
  3. Set Sort by: Class, A to Z
  4. Click Add Level
  5. Set Then by: Score, Largest to Smallest
  6. Click OK
Critical — Never Sort a Single Column Alone

If you only sort one column without selecting all columns, the data relationships are destroyed. John's score gets attached to Mary's name. Always click a single cell (not a whole column) before sorting so Excel selects and sorts all columns together.

Filtering Data

Filtering hides rows that don't match your criteria — without deleting them. Use filtering to show only the data you need at the moment.

Apply and Use Filters
  1. 1 Click any cell in your data table
  2. 2 Go to Data Tab → Filter (or press Ctrl+Shift+L)
  3. 3 Dropdown arrows (▾) appear on each header
  4. 4 Click the arrow on the column you want to filter
  5. 5 Choose your filter type:
    Check/uncheck values — show only selected items
    Number Filters → Greater Than, Less Than, Between, Top 10…
    Text Filters → Contains, Begins With, Does Not Contain…
  6. 6 Click OK — matching rows are shown, others are hidden (row numbers turn blue)
Useful Filter Examples
  • Show only students from "8A"
  • Show scores greater than 80
  • Show expenses between Ksh 1000 and 5000
  • Show all entries containing "Nairobi"
  • Show Top 5 highest sales
Clear Filters

To show all data again:

  • Data Tab → Clear (removes all filters)
  • OR click the filter arrow → Clear Filter From [Column]
  • Clicking Filter again removes all filter dropdowns

Page Setup and Printing

Setting Up Your Page for Printing

Orientation
  • Portrait — taller than wide. Good for most tables
  • Landscape — wider than tall. Good for wide tables with many columns

Page Layout Tab → Orientation

Margins
  • Normal — 2.54 cm, standard for most prints
  • Narrow — 1.27 cm, fits more columns
  • Wide — more white space around content

Page Layout Tab → Margins

Scaling
  • Fit Sheet on One Page — shrinks to fit a single printed page
  • Fit All Columns on One Page — squeezes columns to fit width
  • Page Layout Tab → Scale to Fit group
Print Area
  • Select only the cells you want to print
  • Page Layout Tab → Print Area → Set Print Area
  • Only the selected range prints — not the whole sheet
  • Clear with: Page Layout → Print Area → Clear Print Area

Headers, Footers, and Repeating Rows

Adding a Header or Footer
  1. 1 Go to Insert Tab → Header & Footer
  2. 2 The view switches to Page Layout view
  3. 3 Click in the Header box and type: e.g., "Monthly Sales Report — Castro Odhiambo"
  4. 4 Use the Header & Footer Tools to insert: Page Number, Date, File Name, Sheet Name
  5. 5 Click the Footer box and add page numbers
  6. 6 Click outside the header/footer area to finish
Repeat Header Rows on Every Printed Page

For tables longer than one page, set the header row to repeat:
Page Layout Tab → Print Titles → Rows to Repeat at Top → click row 1
Now "Name, Score, Grade" appears at the top of every printed page automatically.

Print Preview and Printing

How to Preview and Print
  1. 1 Press Ctrl+P (or File → Print)
  2. 2 The Print Preview appears on the right — inspect it carefully
  3. 3 Check: Is everything visible? Are columns cut off? How many pages?
  4. 4 Adjust settings on the left: Printer, Copies, Pages to print, Orientation, Margins, Scaling
  5. 5 When satisfied, click Print
Cybercafé Tip — Always Preview First

At a cybercafé you pay per page printed. A spreadsheet without Print Area set might print 10+ empty pages. Always check Print Preview, set your Print Area, and count the pages before clicking Print.

Saving, File Formats & Protecting Workbooks

Saving Your Workbook

Save (Ctrl+S)

Updates the current file instantly. Use this every few minutes. If the file has not been saved before, it will open the Save As dialog.

Save As (F12)

Save a new copy with a different name, in a different location, or as a different file type. Use to create versions: Budget_Draft.xlsx → Budget_Final.xlsx

Excel File Formats — Choose the Right One

ExtensionFormatWhen to Use
.xlsxExcel Workbook (default)Always use this for normal Excel files (Excel 2007 and newer)
.xlsExcel 97–2003Only if recipient uses very old Excel — otherwise use .xlsx
.csvComma Separated ValuesExport data to other systems (databases, web apps) — formulas and formatting are lost
.pdfPDF DocumentShare spreadsheets that should not be edited — preserves layout
.xlsmMacro-Enabled WorkbookIf the file contains VBA macros (advanced automation)

Protecting Your Workbook and Worksheets

Protection prevents unauthorised users from changing your data, formulas, or structure. This is essential for shared files — payroll, student marks, or business data.

Protect the Entire Workbook with a Password

Password to Open
  1. 1 Click File → Info
  2. 2 Click Protect Workbook → Encrypt with Password
  3. 3 Type a strong password (e.g., Marks@2026!)
  4. 4 Confirm the password → Click OK
  5. 5 Save the file — password required to open next time

Protect a Worksheet (Prevent Editing)

Protect Sheet
  1. 1 Go to Review Tab → Protect Sheet
  2. 2 Enter a password
  3. 3 Choose what users are still allowed to do (e.g., select cells, but not change them)
  4. 4 Click OK — users can view but not edit the sheet
  5. 5 To unprotect: Review Tab → Unprotect Sheet → enter password

Locking Specific Cells (Advanced)

Sometimes you want to protect formulas but allow users to enter data in input cells. Here's how:

Lock Formulas, Unlock Input Cells
  1. 1 Select all cells (Ctrl+A) → Right-click → Format Cells → Protection → check "Locked" → OK
  2. 2 Select only the input cells (cells users should type in) → Right-click → Format Cells → Protection → uncheck "Locked" → OK
  3. 3 Now go to Review Tab → Protect Sheet → enter password → OK
  4. 4 Users can now only type in the unlocked cells — all formulas are protected
Never Forget Your Password

Excel password protection cannot be recovered if you forget the password. Write it down and store it safely. Use a memorable but strong password like Excel@Nairobi2026.
