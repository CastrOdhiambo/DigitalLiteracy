4.1 Opening and Understanding the Excel Interface

4.2 Entering and Editing Data in Excel

Introduction: Data entry is the foundation of Excel. A spreadsheet is only as useful as the data entered correctly.

Learning Objectives: Enter text, numbers, and dates; Edit and modify existing data; Delete or clear cell contents; Copy, cut, and paste efficiently; Navigate quickly between cells.

A. Types of Data in Excel

Data TypeExamplesDescription
Text"John", "Maths", "Total"Labels, names, descriptions
Numbers50, 1000, 3.14Values for calculations
Dates12/02/2026, Jan-15Calendar dates
Times08:30 AM, 14:45Time values

B. Entering Data

Basic Steps: 1. Click the desired cell (e.g., A1). 2. Type your data. 3. Press: Enter → Move down one cell, Tab → Move right one cell, Arrow keys → Move in any direction. Example: Create a simple student record: Cell A1: Type "John" → Press Tab, Cell B1: Type "15" → Press Tab, Cell C1: Type "8A" → Press Enter.

C. Editing Existing Data

Method 1 - Replace All: Click the cell, Type new content (overwrites existing).
Method 2 - Edit Partially: Double-click the cell, OR click the cell then click in the Formula Bar, Make your changes, Press Enter to confirm.
Method 3 - Keyboard Shortcut: Select cell, Press F2 (enters edit mode), Make changes, Press Enter.

D. Deleting and Clearing Data

ActionMethodResult
Delete keySelect cell → Press DeleteClears content only
Clear ContentsHome → Clear → Clear ContentsClears content only
Clear FormatsHome → Clear → Clear FormatsRemoves formatting only
Clear AllHome → Clear → Clear AllRemoves both content and formatting

E. Copying, Cutting, and Pasting

Copy and Paste: Select cell(s), Press Ctrl+C (or right-click → Copy), Select destination, Press Ctrl+V (or right-click → Paste).
Cut and Paste (Move): Select cell(s), Press Ctrl+X (or right-click → Cut), Select destination, Press Ctrl+V.

Keyboard Shortcuts Summary: Copy: Ctrl+C, Cut: Ctrl+X, Paste: Ctrl+V, Undo: Ctrl+Z, Redo: Ctrl+Y.

Common Mistakes to Avoid: Typing numbers as text – Don't use quotes around numbers; Forgetting to save; Accidental overwrites – Use Undo (Ctrl+Z); Inefficient navigation; Not verifying data.

4.3 Basic Formulas and Functions (SUM, AVERAGE, MIN, MAX)

Introduction: Excel's real power lies in automatic calculations. Formulas and functions transform static data into dynamic, interactive spreadsheets.
Key Concepts: Formula: A calculation you create (e.g., =A1+B1); Function: A predefined formula (e.g., =SUM(A1:A10)).

A. Arithmetic Operators

OperatorOperationExample
+Addition=A1+B1
-Subtraction=A2-B2
*Multiplication=A3*B3
/Division=A4/B4

B. Creating Basic Formulas

Example 1 - Addition: =A1+B1. Example 2 - Multiplication with parentheses: =(A2+B2)*C2. Step-by-Step: Click result cell, Type =, Click first cell, Type operator, Click second cell, Press Enter.

C. The SUM Function

Syntax: =SUM(range). Method 1: Type =SUM(A1:A5). Method 2: AutoSum – Select cell below numbers, click Home → AutoSum, verify range, press Enter. Method 3: Type =SUM() and drag to select cells.

D. The AVERAGE Function

Syntax: =AVERAGE(range). Example: =AVERAGE(B2:B10).

E. The MIN and MAX Functions

Syntax: =MIN(range) returns smallest, =MAX(range) returns largest. Examples: =MIN(C5:C20), =MAX(D5:D20).

F. Copying Formulas with Fill Handle

Select cell with formula, hover over bottom-right corner until cursor becomes +, click and drag down or right. Excel automatically adjusts cell references (relative referencing).

Practical Activity 3.1 – Student marks analysis

  1. Open new workbook. Enter data: Student (John, Mary, Peter), Maths(80,75,90), English(90,88,85), Science(85,92,80).
  2. Calculate Totals (Column E): In cell E2 type =SUM(B2:D2), copy down using fill handle.
  3. Calculate Averages (Column F): In cell F2 type =AVERAGE(B2:D2), copy down.
  4. Find Highest Maths Score: In cell B5 type =MAX(B2:B4).
  5. Find Lowest Science Score: In cell D5 type =MIN(D2:D4).
  6. Save as FormulasPractice.xlsx.
Common Mistakes: Forgetting =, Wrong range, Overwriting formulas, Text in calculations (#VALUE!), Dividing by zero (#DIV/0!).

4.6 Sorting and Filtering Data in Excel

Introduction: Large datasets become manageable with sorting and filtering.

A. Basic Sorting

Ascending: Text A→Z, Numbers smallest→largest, Dates oldest→newest. Descending: opposite. Steps: Click any cell in column, Data → Sort A to Z or Sort Z to A.

B. Custom Multiple-Level Sorting

Select data → Data → Sort. First Level: Sort by Class, Values, A to Z. Click Add Level. Second Level: Then by Score, Values, Largest to Smallest. OK.

C. Filtering Data

Apply Filter: Select any cell, Data → Filter. Dropdown arrows appear. Filter by values, Number Filters (Greater Than, etc.), Text Filters (Contains, Begins With).

D. Clearing Filters and Sorts

Data → Clear (in Sort & Filter group).

Practical Activity 6.1

  2. Sort by Score descending (Z to A).

Common Mistakes: Sorting only one column (destroys relationships), Forgetting headers, Not clearing filters.

4.7 Printing and Page Setup in Excel

A. Setting the Print Area

Select range → Page Layout → Print Area → Set Print Area.

B. Page Orientation

Portrait (tall), Landscape (wide tables).

C. Margins

Normal, Narrow, Wide, Custom Margins.

D. Scaling

Fit Sheet on One Page, Fit All Columns, Fit All Rows.

E. Headers and Footers

Insert → Header & Footer. Add page numbers, date, file name.

F. Print Titles (Repeating Rows/Columns)

Page Layout → Print Titles → Rows to repeat at top.

G. Print Preview and Printing

File → Print (Ctrl+P). Save as PDF: Microsoft Print to PDF.

4.8 Protecting and Saving Excel Workbooks

A. Saving Workbooks

Manual Save: File → Save As (F12), Ctrl+S. AutoSave (Office 365) with OneDrive.

B. File Formats

ExtensionFormatUse For
.xlsxExcel WorkbookStandard Excel files (2007+)
.xlsExcel 97-2003Compatibility
.xlsmMacro-enabledFiles with VBA code
.csvComma SeparatedData exchange
.pdfPDF DocumentSharing (non-editable)

C. Protecting with Passwords

File → Info → Protect Workbook → Encrypt with Password. Password to modify: File → Save As → Tools → General Options.

D. Protecting Worksheets

Review → Protect Sheet. Enter password, choose allowed actions.

E. Locking Specific Cells

Select editable cells → Right-click → Format Cells → Protection → Uncheck Locked → Protect Sheet.

F. Mark as Final

File → Info → Protect Workbook → Mark as Final (read-only).

G. Creating Backups

File → Save As → Tools → General Options → Always create backup.
