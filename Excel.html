Section 4.2

Navigation, Selecting Cells & Entering Data

Moving Around the Worksheet

Keyboard Navigation
  • Arrow Keys — move one cell
  • Enter — down one cell
  • Tab — right one cell
  • Ctrl + Home — go to cell A1
  • Ctrl + End — go to last used cell
  • Page Down/Up — move one screen down/up
Mouse Navigation
  • Click any cell to select it
  • Use scroll bars to move up/down/left/right
  • Click sheet tabs (Sheet1, Sheet2) to switch worksheets
  • Right-click sheet tab → Rename to change sheet name

Entering and Editing Data

Entering Data
  1. 1 Click the cell where you want to enter data
  2. 2 Start typing (text, numbers, dates)
  3. 3 Press Enter to confirm and move down, or Tab to move right
Editing an Existing Cell
  1. 1 Double-click the cell OR select it and press F2
  2. 2 Edit the content using arrow keys
  3. 3 Press Enter when finished
Deleting Cell Contents

Select cell(s) and press Delete key to clear content. To remove formatting too, use Home → Clear → Clear All.

AutoFill — Excel's Magic Handle

The fill handle is the small square at the bottom-right corner of any selected cell. Drag it to automatically fill data series.

AutoFill Numbers

Type 1 in cell A1, drag fill handle — copies 1. Type 1 in A1 and 2 in A2, select both, drag → creates sequence 1,2,3,4...

AutoFill Days/Months

Type "Monday" → drag → Tuesday, Wednesday... Type "January" → drag → February, March...

AutoFill Formulas

Enter a formula, then drag the fill handle to copy it to adjacent cells (relative references adjust automatically).

  3. In A2 type Monday, use AutoFill to complete Tuesday–Friday
  4. In B2–B6 type any sales numbers: 500, 700, 300, 900, 600

Section 4.3 — Core Skill

Formulas & Functions: The Heart of Excel

What is a Formula?

A formula is an equation that performs calculations. All formulas in Excel begin with an equals sign (=).

Rule #1

Every formula MUST start with =. If you forget, Excel treats it as plain text and won't calculate.

Basic Arithmetic Operators

OperatorOperationExample FormulaResult
+Addition=5+38
-Subtraction=10-46
*Multiplication=6*742
/Division=20/45

Cell References — The Power of Excel

Instead of typing numbers directly, use cell references. When you change the numbers in cells, formulas update automatically.

AB
1100=A1*2 → 200
250=A1+A2 → 150

SUM Function — Add a Range of Cells

SUM adds all numbers in a range. This is the most commonly used function in Excel.

Syntax: =SUM(number1, number2, ...) or =SUM(range)
  1. 1 Click the cell where you want the total
  2. 2 Type =SUM(
  3. 3 Select the cells to add (e.g., B2:B10)
  4. 4 Type ) and press Enter
AutoSum Shortcut

Click the AutoSum button Σ on the Home tab or press Alt+= — Excel automatically detects the range to sum.

AVERAGE, MAX, MIN, COUNT Functions

AVERAGE

=AVERAGE(range) — calculates the mean of numbers.

Example: =AVERAGE(B2:B10)
MAX

=MAX(range) — finds the largest number.

Example: =MAX(B2:B10)
MIN

=MIN(range) — finds the smallest number.

Example: =MIN(B2:B10)
COUNT

=COUNT(range) — counts cells containing numbers.

Example: =COUNT(A2:A50)

IF Function — Make Decisions

IF returns one value if a condition is TRUE, and another value if FALSE. It is used for pass/fail, above/below target, etc.

Syntax: =IF(logical_test, value_if_true, value_if_false)
  1. 1 Type =IF(
  2. 2 Enter condition (e.g., B2>=50)
  3. 3 Type comma, then result if true (e.g., "Pass")
  4. 4 Type comma, then result if false (e.g., "Fail")
  5. 5 Type ) and press Enter
ABC
1NameScoreResult
2John75=IF(B2>=50,"Pass","Fail") → Pass
3Mary42=IF(B3>=50,"Pass","Fail") → Fail

Section 4.4

Formatting Cells — Make Data Professional

Number Formatting — Currency, Percentage, Decimals

Currency (Ksh)

Select cells → Home Tab → Number group → click Currency ($ symbol) → Kenya uses Ksh. Use Accounting format for tables.

Percentage

Select cells → Click % symbol. Example: 0.25 becomes 25%.

Increase/Decrease Decimals

Use .0→.0 buttons to show 2 decimal places (e.g., 1000.00).

Cell Borders & Fill Color

  • Borders: Home Tab → Font group → Borders dropdown → choose All Borders
  • Fill Color: Paint bucket icon → choose background colour for headers
  • Font Color: Letter A with colour bar → change text colour
  • Merge & Center: Select multiple cells → Home Tab → Merge & Center (creates a title over multiple columns)

Conditional Formatting — Colour Based on Value

Conditional Formatting automatically changes cell appearance based on its value. Perfect for highlighting low scores, high sales, or overdue dates.

Highlight cells above a threshold
  1. 1 Select your data range
  2. 2 Home Tab → Conditional Formatting
  3. 3 Highlight Cells Rules → Greater Than
  4. 4 Enter value (e.g., 70) and choose format (Green Fill)
4 Format the Marksheet

  3. Format Total column as Currency (Ksh)
  4. Add All Borders to the entire table
  5. Apply Conditional Formatting to Average column: Highlight averages below 50 in red

Sorting & Filtering Data

Sorting Data

Sort a Column (A to Z or Smallest to Largest)
  1. 1 Select any cell in the column you want to sort
  2. 2 Data Tab → Sort A to Z (ascending) or Sort Z to A (descending)
  3. 3 For multiple-level sorts, click Sort and add levels

Filtering — Show Only What You Need

Apply a Filter
  1. 1 Select your data range (including headers)
  2. 2 Data Tab → Filter button (funnel icon)
  3. 3 Dropdown arrows appear on headers — click to filter by specific values
Clear Filter: Click the filter funnel icon again, or use Data → Clear.

Printing in Excel — Page Setup & Print Area

Set Print Area & Page Layout

Set Print Area
  1. 1 Select the cells you want to print
  2. 2 Page Layout Tab → Print Area → Set Print Area
Fit Sheet on One Page
  1. 1 Page Layout Tab → Width → 1 page
  2. 2 Height → 1 page
  3. 3 Check Print Preview (Ctrl+P)
Always Preview Before Printing

Press Ctrl+P to see exactly what will print. Adjust margins and scaling as needed.

Section 4.8 — Real-World Project

Complete Monthly Budget Project

Project: Create a Personal Monthly Budget

This project combines everything you've learned: data entry, formulas, formatting, charting, and printing.

Step 1 — Set Up the Budget Structure
  1. 1 Open new blank workbook. Rename Sheet1 to "January"
  2. 2 In row 1: A1: Category, B1: Budget (Ksh), C1: Actual (Ksh), D1: Difference, E1: Status
  3. 3 In column A, list categories: Rent, Groceries, Transport, Airtime, Electricity, Water, Entertainment, Savings, Miscellaneous
Step 2 — Enter Sample Numbers
  1. 1 Enter realistic Budget amounts (e.g., Rent: 8000, Groceries: 5000, Transport: 2000, Airtime: 1000)
  2. 2 Enter Actual amounts (some higher, some lower than budget)
Step 3 — Add Formulas
  1. 1 In D2 (Difference): =C2-B2 → drag down (positive means under budget, negative means over)
  2. 2 In E2 (Status): =IF(D2>=0,"Under","Over") → drag down
  3. 3 At bottom, add TOTAL row: =SUM(B2:B10) for Budget Total, same for Actual and Difference
Step 4 — Format the Budget
  1. 1 Format all currency numbers as Kenyan Shillings (Number → Currency, symbol Ksh)
  2. 2 Bold headers, add borders, use conditional formatting to highlight negative differences in red
  3. 3 Create a Pie Chart showing Actual spending by category (Insert → Pie Chart)
Step 5 — Finalise
  1. 1 Set Print Area to include the table and chart
  2. 2 Fit to one page wide
  3. 3 Add a header with your name and "Monthly Budget - January"
  4. 4 Save as My_Monthly_Budget
8 Build Your Own Budget
  1. Follow all steps above using YOUR own expense categories and numbers
  2. Create a pie chart of your Actual expenses
  3. Use IF to show "Under Budget" or "Over Budget" for each category
  4. Preview and ensure it fits on one page
  5. Save as YourName_Budget
