Introduction
Microsoft Word (MS Word) is a word processing application used to create, edit, format, and print documents.

It is commonly used for:

  • Letters
  • Reports
  • Assignments
  • CVs
  • Memos
  • Examination papers

Learning MS Word is one of the most important digital skills because it is widely used in schools, offices, businesses, and government institutions.

In this chapter, you will learn how to:

  • Open Microsoft Word
  • Understand the Word interface
  • Identify important tools
  • Navigate the workspace confidently

This chapter is fully practical.

3.1 Opening and Understanding the MS Word Interface

A. Opening Microsoft Word

There are several ways to open MS Word.

Method I: Using the Start Menu
Follow these steps:

  1. Click the Start Button.
  2. Scroll through the list of programs.
  3. Locate Microsoft Office or Microsoft Word.
  4. Click Microsoft Word.
  5. Wait for the application to open.

Method II: Using Search

  1. Click the Start Button.
  2. Type Word in the search bar.
  3. Click Microsoft Word from the search results.

Method III: Using a Desktop Shortcut

  1. Locate the Word icon on the desktop.
  2. Double-click the icon.

When Word opens, you may see:
A Start Screen • Templates • Blank Document option
Click Blank Document to begin typing.

B. Understanding the MS Word Interface

When you open a blank document, you will see the Word workspace. Understanding the interface helps you work efficiently.

Main Parts of the MS Word Window

  • I. Title Bar – Located at the top of the screen. It shows: The name of the document, The name of the application (Word). Example: Document1 - Word
  • II. Quick Access Toolbar – Located at the top-left corner. It contains frequently used commands such as: Save, Undo, Redo.
  • III. The Ribbon – The Ribbon is the main tool area of MS Word. It contains tabs such as: Home, Insert, Design, Layout, References, Review, View. Each tab contains groups of related tools. Example: The Home Tab contains: Font tools, Paragraph tools, Clipboard tools.
  • IV. Document Area – This is the large white space where you type your work. It is also called the Workspace.
  • V. Status Bar – Located at the bottom of the screen. It shows: Page number, Word count, Language, Zoom level.
  • VI. Scroll Bars – Located on the right side (vertical) and bottom (horizontal). Used to move through the document.
  • VII. Ruler – Located at the top and left side (if enabled). Used for: Setting margins, Aligning text, Adjusting indents.

Practical Activity 1: Exploring the Interface
Follow these steps:

  1. Open Microsoft Word.
  2. Click Blank Document.
  3. Identify the Title Bar.
  4. Click the Home Tab.
  5. Click the Insert Tab.
  6. Locate the Status Bar at the bottom.
  7. Close the document without saving.

Repeat until you can identify each part without assistance.

C. Understanding Tabs and Groups

Each tab contains groups of tools.
Example from the Home Tab:
Clipboard Group (Copy, Paste, Cut)
Font Group (Bold, Italic, Font Size)
Paragraph Group (Alignment, Bullets)

When you click a tab, different tools appear.

Practical Activity 2

  1. Open Word.
  2. Click the Home Tab.
  3. Locate the Bold button (B).
  4. Locate the Font Size box.
  5. Click the Insert Tab and identify the Table option.

Why Understanding the Interface Is Important
- Increases speed
- Reduces confusion
- Improves confidence
- Makes formatting easier

3.2 Creating, Saving, and Opening Documents

Creating and saving documents properly is one of the most important skills in Microsoft Word. If you do not save your work correctly, you may lose it. In this section, you will learn how to:
Create a new document, Save a document for the first time, Save changes to an existing document, Open a saved document. This section must be practiced carefully.

A. Creating a New Document

When you open Word, you may already see a blank document. If not, follow these steps:

Steps to Create a New Document

  1. Open Microsoft Word.
  2. Click Blank Document.
  3. A new typing page will appear.
  4. Place the cursor in the document area.
  5. Begin typing your text.

You have now created a new document.

Practical Activity 1

  1. Open MS Word.
  2. Create a blank document.
  3. Type the following: My name is ____________. I am learning digital competency. Microsoft Word is useful for creating documents.
  4. Do not close the document yet.

B. Saving a Document for the First Time

Saving allows the computer to store your work permanently.

Steps to Save a New Document

  1. Click File (top-left corner).
  2. Click Save As.
  3. Choose the location (e.g., Documents or Desktop).
  4. Click the selected location.
  5. Type the file name (e.g., Word Practice).
  6. Click Save.
  7. The file is now saved.
Important Note: If you do not save your work and the computer shuts down unexpectedly, your work may be lost. Always save regularly.

C. Saving Changes to an Existing Document

After making changes, you must save again.

Method I: Using the Save Button
Click the Save icon (disk symbol) in the Quick Access Toolbar.

Method II: Using Keyboard Shortcut
Press Ctrl + S on the keyboard. This updates the document with recent changes.

Good Practice Tip: Press Ctrl + S every few minutes while working.

D. Opening an Existing Document

If you have already saved a document, you can open it again later.

Method I: Opening from Word

  1. Open Microsoft Word.
  2. Click File.
  3. Click Open.
  4. Click Browse.
  5. Navigate to the folder where the file was saved.
  6. Double-click the file name.

The document will open.

Method II: Opening from File Explorer

  1. Open File Explorer.
  2. Navigate to the folder containing the file.
  3. Double-click the file.

Word will open automatically.

E. Understanding "Save As"

"Save As" is used when: Saving a file for the first time, Saving a copy with a different name, Saving in a different location.
Example: You have a file named Report. Using Save As allows you to create another file named Final Report.

Practical Activity 2

  1. Create a new document.
  2. Type a short paragraph about your school or workplace.
  3. Save the document as My School Information.
  4. Close the document.
  5. Re-open it using File Explorer.
  6. Add one more sentence.
  7. Press Ctrl + S to save changes.
Common Mistakes to Avoid: Closing a document without saving, Saving in the wrong location, Forgetting the file name, Not updating changes.

3.3 Basic Text Formatting (Fonts, Size, Bold, Italics, Alignment)

Typing a document is not enough. A professional document must be well formatted. Formatting means changing the appearance of text to make it: Clear, Attractive, Easy to read, Professional.

In this section, you will learn how to: Change font type, Change font size, Use Bold, Italic, and Underline, Change text color, Align text properly. All formatting tools are found in the Home Tab.

A. Selecting Text Before Formatting

Steps to Select Text:

  1. Place the mouse pointer at the beginning of the text.
  2. Click and hold the left mouse button.
  3. Drag over the text.
  4. Release the mouse button. The selected text will appear highlighted.

B. Changing Font Type

Examples of fonts: Arial, Times New Roman, Calibri.

Steps:

  1. Select the text.
  2. Go to the Home Tab.
  3. Click the Font drop-down menu.
  4. Choose a font style.

C. Changing Font Size

Steps:

  1. Select the text.
  2. Click the Font Size box in the Home Tab.
  3. Choose a size (e.g., 12, 14, 16).

Common practice: Body text: Size 12, Headings: Size 14-18.

D. Using Bold, Italic, and Underline

Bold (B) makes text darker and stronger. Italic (I) slants text. Underline (U) draws a line under text.

Steps: Select the text. Click B, I, or U in the Home Tab.

E. Changing Text Color

Steps: Select the text. Click the Font Color icon (A with a colored line). Choose a color. Avoid using too many colors in formal documents.

F. Text Alignment

There are four types: I. Align Left, II. Center, III. Align Right, IV. Justify.

Steps to Change Alignment: Select the text. Go to the Home Tab. Click one of the alignment buttons in the Paragraph group.

When to Use Alignment: Left Align → Normal paragraphs, Center → Titles and headings, Right Align → Dates and signatures, Justify → Reports and formal documents.

Practical Activity 1

  1. Open MS Word.
  2. Type the following: DIGITAL SKILLS TRAINING, My name is ______. I am learning how to format documents professionally.
  3. Select the first line. Make it: Bold, Font size 16, Center aligned.
  4. Select the paragraph. Change: Font to Arial, Font size to 12, Justify alignment.
  5. Save the document.

G. Using Format Painter (Optional Advanced Skill)

Format Painter copies formatting from one text to another. Steps: Select formatted text. Click Format Painter in the Home Tab. Select the new text. The formatting will be copied.

Common Mistakes to Avoid: Formatting without selecting text, Using too many fonts in one document, Using too many colors, Using very large font sizes unnecessarily.

Mini Practice Challenge
Create a short document with: A centered heading (bold, size 18), A left-aligned paragraph, One sentence in italics, One word underlined. Save as: Formatting Practice.

3.4 Paragraph Formatting (Line Spacing, Indentation, Bullets & Numbering)

After learning how to format text, the next step is formatting paragraphs. Paragraph formatting helps make documents: Clear, Organized, Easy to read, Professional.

A. Line Spacing

Steps: Select the paragraph. Go to Home Tab → Paragraph group. Click Line and Paragraph Spacing. Choose the spacing (e.g., 1.0, 1.5, 2.0). Practical Tip: School reports → 1.5 or 2.0 spacing, Business letters → 1.15 or 1.5 spacing.

B. Paragraph Indentation

Indentation moves text inward from the margin. Types of Indentation: Left Indent, Right Indent, First Line Indent, Hanging Indent. Steps to Apply Indentation: Select the paragraph. Go to Home Tab → Paragraph group. Click the Increase Indent or Decrease Indent buttons. For advanced settings: Click the small arrow in the bottom-right corner of the Paragraph group, adjust Left, Right, or Special settings, Click OK.

C. Bullets and Numbering

Using Bullets: Select the lines of text. Go to Home Tab → Paragraph group. Click Bullets. Choose a bullet style. Using Numbering: Select the lines of text. Go to Home Tab → Paragraph group. Click Numbering. Choose a numbering style. Practical Tip: Bullets → Unordered lists, Numbering → Steps or sequences.

D. Aligning and Spacing Paragraphs

Use Left, Center, Right, or Justify for paragraph alignment (covered in 3.3). Use Spacing Before and After to add space between paragraphs. Steps: Select paragraph, Click the small arrow in the Paragraph group, Adjust Spacing Before and After, Click OK.

Practical Activity 1

  1. Open a new Word document.
  2. Type the following: Steps to Prepare a Healthy Breakfast, 1. Wash fruits and vegetables, 2. Prepare cereal or porridge, 3. Serve juice or water.
  3. Select the paragraph title and make it Bold and Centered.
  4. Apply 1.5 line spacing to the list.
  5. Apply Numbering to the steps.
  6. Indent the list by using the Increase Indent button.
  7. Save the document as Paragraph Practice.
Common Mistakes to Avoid: Forgetting to select the paragraph before formatting, Using inconsistent spacing or indentation, Using bullets for steps that require numbering, Leaving paragraphs too close together.

3.5 Inserting and Formatting Tables

Tables are used to organize information neatly in rows and columns. They are useful for: Schedules, Marksheets, Financial data, Lists, Reports.

A. Inserting a Table

Steps: Open a Word document. Click the Insert Tab. Click Table. Move the cursor over the grid to select the number of rows and columns. Click to insert the table. Example: 4 columns × 5 rows table for a weekly schedule.

B. Adding or Removing Rows and Columns

Adding Rows: Place the cursor in the row above or below where you want a new row. Right-click → Insert → Choose Insert Rows Above or Insert Rows Below. Adding Columns: Place the cursor in the column next to where you want a new column. Right-click → Insert → Choose Insert Columns to the Left or Right. Deleting Rows or Columns: Select the row or column. Right-click → Delete Cells → Choose Delete Entire Row or Column.

C. Merging and Splitting Cells

Merging Cells: Select the cells to merge. Right-click → Merge Cells. Use: To create a heading that spans multiple columns. Splitting Cells: Select the cell to split. Right-click → Split Cells → Specify number of rows or columns → Click OK. Use: To divide a single cell into multiple sections.

D. Formatting Tables

Applying Table Styles: Click anywhere in the table. Go to Table Design Tab (appears automatically). Select a Table Style from the gallery. Adjust shading, borders, and colors. Aligning Text Inside Cells: Select the cells. Go to Layout Tab → Alignment group. Choose alignment: top-left, center, bottom-right, etc. Adjusting Column Width and Row Height: Hover the mouse over column borders → drag to resize. Hover over row borders → drag to adjust height.

Practical Activity 1

  1. Open a new Word document.
  2. Insert a table with 4 columns × 5 rows.
  3. Type the following headings in the first row: Day | Task | Start Time | End Time.
  4. Fill in three rows with your daily schedule.
  5. Apply bold to the heading row.
  6. Center align the text in the heading row.
  7. Apply a Table Style of your choice.
  8. Save the document as Table Practice.
Common Mistakes to Avoid: Forgetting to select the correct number of rows or columns initially, Not saving the table after formatting, Overcomplicating table design in formal documents, Ignoring text alignment inside cells.

3.6 Inserting Pictures, Shapes, and WordArt

Visual elements make documents engaging and easier to understand. You can insert pictures from your computer, insert shapes, use WordArt for decorative text. All these tools are found in the Insert Tab.

A. Inserting Pictures

Steps: Open a Word document. Click the Insert Tab. Click Pictures → Choose This Device. Browse and select the image you want. Click Insert. The picture will appear in your document. Adjusting the Picture: Click on the picture. Use handles on the corners to resize proportionally. Go to Picture Format Tab to: Change position (Wrap Text → Square, Tight, Behind Text, etc.), Apply Picture Styles (border, shadow, effects), Crop the picture if needed.

B. Inserting Shapes

Steps: Go to the Insert Tab → Shapes. Select a shape (rectangle, circle, arrow, etc.). Click and drag on the document to draw the shape. Use Shape Format Tab to change color, outline, or effects.

C. Using WordArt

Steps: Go to the Insert Tab → WordArt. Select a style. Type your text in the box that appears. Adjust size, font, and color as desired. Use Drawing Tools to rotate or position the WordArt.

Practical Activity 1

  1. Open a new Word document.
  2. Type the heading: My Favorite Hobby.
  3. Insert a relevant picture from your computer.
  4. Insert a rectangle shape behind the heading and change its color.
  5. Use WordArt for the heading text.
  6. Resize and align all elements neatly.
  7. Save the document as Visual Practice.
Tips for Using Visual Elements: Keep images relevant to your content, Avoid overloading the document with too many shapes or pictures, Use WordArt sparingly in formal documents, Always save your document after inserting visuals.
Common Mistakes to Avoid: Stretching images disproportionally, Placing pictures in a way that blocks text, Using too many decorative fonts for formal work, Forgetting to save after inserting images.

3.7 Page Layout and Printing (Margins, Orientation, Header/Footer, Print Preview)

Page layout is about how the document appears on the page. Proper layout ensures that documents are readable, well-organized, and ready for printing.

A. Setting Margins

Margins are the blank spaces around the edges of the page. Steps: Go to the Layout Tab. Click Margins. Choose a preset margin (Normal, Narrow, Moderate) or click Custom Margins. Adjust Top, Bottom, Left, and Right margins if needed. Click OK. Tip: Normal margins (2.54 cm) are standard for most documents.

B. Changing Page Orientation

Page orientation is how the page is positioned: Portrait → Vertical (default), Landscape → Horizontal. Steps: Go to the Layout Tab → Orientation. Select Portrait or Landscape. Use Cases: Portrait → Letters, essays; Landscape → Wide tables or charts.

C. Inserting Header and Footer

Headers and footers appear at the top or bottom of each page. They often include: Page numbers, Titles, Dates. Steps: Go to the Insert Tab. Click Header → Choose a style. Type the desired text. Click Close Header and Footer. Repeat the same steps for Footer. Adding Page Numbers: Go to Insert Tab → Page Number. Choose location (Top or Bottom of page). Select numbering style.

D. Using Print Preview

Print Preview allows you to see how the document will look when printed. Steps: Click File → Print. Check the Preview Pane on the right. Make adjustments if needed (orientation, margins, page range). Select printer settings (Printer, Copies). Click Print to print the document.

Practical Activity 1

  1. Open a Word document.
  2. Set Margins to Normal.
  3. Change Orientation to Landscape.
  4. Insert a Header with the title: "My Project Report".
  5. Insert a Footer with page numbers.
  6. Preview the document using Print Preview.
  7. Save the document as PageLayoutPractice.
Tips for Page Layout: Always check margins before printing, Use headers/footers to identify documents, Print a preview before sending to printer, Use landscape orientation only for wide tables or charts.
Common Mistakes to Avoid: Forgetting to set margins, causing text to cut off, Printing without previewing, Placing page numbers incorrectly, Using too many decorative headers in formal documents.
