Section 3.1

Opening Word & Understanding the Interface

Opening Microsoft Word

There are three ways to open Microsoft Word. Learn all three — each is useful in different situations.

Method 1 — Start Menu
  1. Click the Start Button
  2. Scroll and find Microsoft Word
  3. Click it to open
Method 2 — Search
  1. Click Start
  2. Type Word in the search bar
  3. Click Microsoft Word in results
Method 3 — Desktop Shortcut
  1. Find the Word icon on the Desktop
  2. Double-click it to open
Method 4 — File Explorer
  1. Navigate to a .docx file
  2. Double-click the file
  3. Word opens with that document
When Word Opens

You will see a Start Screen with templates. Click Blank Document to open a fresh, empty page and begin typing.

Section 3.2

Creating, Saving & Opening Documents

Creating a New Document

From the Start Screen
  1. Open Word
  2. Click Blank Document
  3. Begin typing immediately
While Word is Already Open
  1. Press Ctrl+N
  2. A new blank document opens instantly

OR: Click File → New → Blank Document

Saving a Document — Correctly, Every Time

Saving stores your work permanently. If you don't save and the power goes off — your work is gone. Save early and save often.

Saving for the First Time (Save As)

Save As — First Save
  1. 1 Click File tab (top-left corner)
  2. 2 Click Save As
  3. 3 Click Browse or choose a location (Desktop, Documents, etc.)
  4. 4 Type a clear file name (e.g., My_CV_2026)
  5. 5 Confirm the file type is Word Document (.docx)
  6. 6 Click Save

Saving Changes to an Existing Document

Fastest Method

Press Ctrl+S at any time

Do this every few minutes while working. Make it a habit!

Using the Toolbar

Click the 💾 Save icon in the Quick Access Toolbar (top-left)

Save As vs Save — Know the Difference

Save (Ctrl+S) — Updates the current file in its current location. Use this constantly.
Save As — Saves a copy with a new name or in a new location. Use when saving for the first time or creating a version (e.g., turning Report_Draft into Report_Final).

Opening an Existing Document

From Inside Word
  1. Click File → Open
  2. Click Browse
  3. Navigate to the folder
  4. Double-click the file name
From File Explorer
  1. Open File Explorer
  2. Navigate to where the file is saved
  3. Double-click the .docx file
  4. Word opens it automatically
Recent Documents — Fastest Way

Click File → Open → Recent to see a list of your recently opened documents. No searching needed!

Section 3.3

Text Formatting — Font, Size, Bold, Alignment & More

Step 1 — Always Select Text First

Before applying any formatting, you must first select (highlight) the text you want to change. Formatting without selecting first has no effect, or will only format new text you type.

Mouse Selection
  1. Click at the start of your text
  2. Hold the left button and drag over the text
  3. Release — the text turns blue/highlighted
Keyboard Selection
  1. Click where you want to start
  2. Hold Shift and press arrow keys or End
  3. To select ALL: press Ctrl+A

Changing Font Type and Size

How to Change Font
  1. 1 Select the text you want to change
  2. 2 Go to the Home Tab
  3. 3 Click the Font drop-down box (shows the current font name)
  4. 4 Type or scroll to choose a font (e.g., Arial, Times New Roman)
  5. 5 Click the font name — your text changes immediately

Changing Font Size

How to Change Size
  1. 1 Select the text
  2. 2 Click the Font Size box next to the font name
  3. 3 Type a size number or click to choose one
  4. 4 Press Enter
Standard Font Sizes for Professional Documents
  • Body text (paragraphs): Size 11 or 12
  • Subheadings: Size 13 or 14
  • Main headings: Size 16–18
  • Title / Cover page: Size 20–28
  • Headers and footers: Size 9 or 10

Bold, Italic, Underline — and When to Use Each

StyleShortcutEffectBest Used For
Bold (B)Ctrl+BMakes text darker and heavierHeadings, key words, names
Italic (I)Ctrl+ISlants text to the right like thisBook titles, foreign words, emphasis
Underline (U)Ctrl+UDraws a line under textImportant terms, links
StrikethroughHome Tab → Font groupDraws a line through textShowing deleted or changed content
HighlightHome Tab → Font groupColoured background on textMarking important sections
Formatting Mistakes to Avoid
  • Don't use more than 2 font styles in one document
  • Avoid Bold for entire paragraphs — reserve it for key words only
  • Don't mix colours randomly in formal documents
  • Never use ALL CAPS for long paragraphs — hard to read

Text Alignment — Left, Centre, Right, Justify

Alignment controls how text lines up horizontally on the page. Always use the correct alignment for each part of your document.

Ctrl+L — Left Align (default)
This text is aligned to the left margin. This is the standard alignment for normal paragraphs in letters and reports.
Ctrl+E — Centre Align
This text is centred. Used for headings, titles, and certificates.
Ctrl+R — Right Align
Used for dates, addresses on the right, and signatures.
Ctrl+J — Justify
This text is justified — stretched to align with both left and right margins evenly. This is used in formal reports, books, and newspapers for a clean professional look.

Format Painter — Copy Formatting Instantly

Format Painter is one of Word's most powerful time-saving tools. It copies the formatting (font, size, bold, colour, alignment) from one piece of text and applies it to another — without retyping anything.

How to Use Format Painter
  1. 1 Select the text that has the formatting you want to copy
  2. 2 Go to Home Tab → Clipboard group
  3. 3 Click the Format Painter button (looks like a paintbrush 🖌️)
  4. 4 Your cursor becomes a paintbrush
  5. 5 Select the text you want to apply the formatting to
  6. 6 Release — the formatting is copied instantly!
Pro Tip — Apply to Multiple Places

Double-click the Format Painter button to keep it active and apply the same formatting to many different sections. Press Esc when done.

Section 3.4

Paragraph Formatting — Spacing, Indents, Lists

Line Spacing

Line spacing controls the amount of vertical space between lines in a paragraph. Correct line spacing makes documents much easier to read.

How to Change Line Spacing
  1. 1 Select the paragraph(s)
  2. 2 Go to Home Tab → Paragraph group
  3. 3 Click the Line and Paragraph Spacing icon (lines with arrows)
  4. 4 Choose your spacing: 1.0, 1.15, 1.5, 2.0, or click Line Spacing Options for more
SpacingUse For
1.0 (Single)Compact content — tables, headers, footers
1.15Default Word spacing — modern business documents
1.5School assignments, reports — easy to read & annotate
2.0 (Double)Academic essays, formal reports that require teacher marking

Paragraph Indentation

Indentation moves text inward from the margin. It is used to show structure, hierarchy, and organisation in a document.

TypeWhat It DoesHow to Apply
Left IndentMoves the entire paragraph from the left marginHome Tab → Increase Indent button, or drag the left indent marker on the Ruler
Right IndentMoves the entire paragraph from the right marginDrag the right indent marker on the Ruler
First Line IndentIndents only the first line — like a book paragraphParagraph dialog → Special → First Line
Hanging IndentFirst line is at the margin; rest are indented. Used in bibliographiesParagraph dialog → Special → Hanging
Quick Indent Shortcut

Press Tab at the start of a paragraph to indent the first line. Press Shift+Tab to remove the indent.

Bullets and Numbered Lists

Lists make information clear and scannable. Always choose the right type of list for your content.

Bullet Lists — Unordered
  • Used when the order of items does NOT matter
  • Example: List of skills on a CV
  • Example: Equipment needed for a task

How: Home Tab → Paragraph group → click Bullets button (•) → choose a style

Numbered Lists — Ordered
  • Used when the ORDER of items matters
  • Example: Steps in a procedure
  • Example: Instructions / recipes

How: Home Tab → Paragraph group → click Numbering button (1.) → choose a style

Styles — Professional Headings in One Click

Word's built-in Styles (found in the Home Tab) apply consistent heading formatting throughout your document with a single click. This is how professional documents are formatted.

  • Heading 1 — Main section title (largest)
  • Heading 2 — Subsection title
  • Heading 3 — Sub-subsection
  • Normal — Body paragraph text
Using Styles Also Builds Your Table of Contents

When you use Heading 1, Heading 2 etc., Word can automatically generate a Table of Contents for long documents. This is used in reports, theses, and books.

Section 3.5

Inserting and Formatting Tables

What Tables Are Used For

Tables organise information into rows and columns, making data clear and easy to compare. You will use tables constantly in professional documents.

Schedules & timetables Marksheets & grades Budgets & expenses CV skills section Inventory lists Invoices & quotes

Inserting a Table

Insert a Table — Step by Step
  1. 1 Click where you want the table in your document
  2. 2 Go to Insert Tab
  3. 3 Click Table
  4. 4 Move the mouse over the grid — highlight the number of columns × rows you need
  5. 5 Click to insert the table
Moving Between Cells

Press Tab to move to the next cell. Press Shift+Tab to go back. Press Tab from the last cell to automatically add a new row.

Editing Tables — Add, Delete, Merge

Adding Rows & Columns
  • Right-click inside the table → Insert
  • Choose: Insert Rows Above / Below
  • Choose: Insert Columns Left / Right
  • OR: Click the + button that appears outside the table
Deleting Rows & Columns
  • Select the row or column
  • Right-click → Delete Cells
  • Choose: Delete Entire Row or Column
Merging Cells
  • Select the cells to merge
  • Right-click → Merge Cells
  • Used for table titles that span multiple columns
Splitting Cells
  • Click inside the cell to split
  • Right-click → Split Cells
  • Enter number of rows/columns → OK

Formatting Tables Professionally

  • Table Styles: Click inside the table → Table Design Tab appears automatically → choose a style from the gallery
  • Bold the header row: Select the top row → press Ctrl+B
  • Centre text in cells: Select cells → Layout Tab → Alignment group → choose Centre
  • Resize columns: Hover the mouse over a column border until you see the resize cursor, then drag
  • Add shading: Select cells → Table Design Tab → Shading → choose a colour
  • Change borders: Table Design Tab → Borders → choose style, width, and colour

Section 3.6

Inserting Pictures, Shapes & WordArt

Inserting a Picture from Your Computer

Steps to Insert an Image
  1. 1 Click where you want the image in your document
  2. 2 Go to Insert Tab
  3. 3 Click Pictures → This Device
  4. 4 Browse to find your image file
  5. 5 Click the image → Click Insert
  6. 6 The picture appears in your document

Adjusting the Picture After Inserting

  • Resize: Click the image — drag the corner handles (always use corners to keep the image proportional)
  • Wrap Text: Click image → Picture Format Tab → Wrap Text → choose Square or Tight to let text flow around the image
  • Move freely: Set Wrap Text to In Front of Text then drag anywhere
  • Crop: Picture Format Tab → Crop → drag the edges to crop the image
  • Picture Styles: Picture Format Tab → choose a frame, shadow, or 3D effect
  • Brightness/Contrast: Picture Format Tab → Corrections → adjust brightness or contrast
Image Mistakes to Avoid
  • Never drag from the side handles — this stretches and distorts the image. Always use corners.
  • Don't use pictures that block your text and make the document unreadable
  • Keep images relevant to the document's purpose

Inserting Shapes

Shapes are used to create diagrams, highlight information, or decorate a document. Shapes include rectangles, circles, arrows, lines, callout boxes, and more.

Steps to Insert a Shape
  1. 1 Go to Insert Tab → Shapes
  2. 2 Click the shape you want (e.g., Rectangle, Circle, Arrow)
  3. 3 Click and drag on the document to draw the shape
  4. 4 Use the Shape Format Tab to change the fill colour, outline, and effects
  5. 5 Right-click the shape → Add Text to type inside it

Using WordArt for Decorative Headings

WordArt creates stylised, decorative text — useful for certificates, posters, and document covers. Use WordArt sparingly — never in formal professional documents like CVs or business letters.

Steps to Insert WordArt
  1. 1 Go to Insert Tab → WordArt
  2. 2 Choose a style from the gallery
  3. 3 A text box appears — type your text
  4. 4 Use the Shape Format Tab to change colour, size, rotation, and text effects

Section 3.7

Page Layout, Headers, Footers & Printing

Setting Page Margins

Margins are the blank spaces around all four edges of the page. Setting margins correctly ensures your document prints neatly without text being cut off.

How to Set Margins
  1. 1 Go to the Layout Tab
  2. 2 Click Margins
  3. 3 Choose a preset: Normal (2.54 cm all round) is standard for most documents
  4. 4 For custom margins, click Custom Margins, enter measurements, click OK
PresetMarginsBest For
Normal2.54 cm all sidesMost documents — letters, reports, assignments
Narrow1.27 cm all sidesTables or wide content that needs more space
Moderate2.54 top/bottom, 1.91 sidesBalanced look with a bit more content space
Wide2.54 top/bottom, 5.08 sidesShort text with lots of white space

Page Orientation & Paper Size

Orientation
  • Portrait (vertical) — default for most documents: letters, CVs, essays, reports
  • Landscape (horizontal) — used for wide tables, charts, certificates, menus

Layout Tab → Orientation → Portrait or Landscape

Paper Size
  • A4 (21 × 29.7 cm) — standard for almost all printing in Kenya
  • Letter — used in USA — different size, avoid in Kenya
  • A3 — large format for posters and diagrams

Layout Tab → Size

Inserting Headers and Footers

Headers appear at the top of every page. Footers appear at the bottom of every page. They are used to add page numbers, document titles, dates, and author names.

How to Add a Header
  1. 1 Go to Insert Tab
  2. 2 Click Header
  3. 3 Choose a style from the gallery
  4. 4 Type your header text (e.g., document title, your name)
  5. 5 Click Close Header and Footer (or double-click outside the header area)
How to Add Page Numbers
  1. 1 Go to Insert Tab
  2. 2 Click Page Number
  3. 3 Choose location: Top of Page, Bottom of Page, Page Margins
  4. 4 Choose a numbering style
  5. 5 Numbers are added automatically to every page
First Page Without Page Number

If you don't want a page number on the first page (e.g., cover page), double-click inside the Header or Footer area, then check the box "Different First Page" in the Header & Footer Tools.

Printing Your Document

How to Print Correctly
  1. 1 Press Ctrl+P — or — click File → Print
  2. 2 Check the Print Preview on the right — does your document look correct?
  3. 3 Select your Printer from the dropdown
  4. 4 Set Number of Copies
  5. 5 Choose which pages to print: All, or a specific range (e.g., 1-3)
  6. 6 Choose One-sided or Two-sided printing if your printer supports it
  7. 7 Click Print
Always Preview Before Printing

At a cybercafé you pay per page. Always check Print Preview first. Mistakes waste money and paper. Verify: margins look right, nothing is cut off, the page count is what you expected.

Saving as PDF

PDF format preserves your document's appearance on any device. Send CVs and formal documents as PDF — they cannot be accidentally changed by the recipient.

Save as PDF
  1. 1 Click File → Save As
  2. 2 Choose your save location
  3. 3 In the "Save as type" dropdown, choose PDF (*.pdf)
  4. 4 Type the file name → Click Save

Section 3.8 — Real-World Project

Writing a Professional CV in Word

What Is a CV?

A CV (Curriculum Vitae) is a document that summarises your personal information, education, work experience, and skills. You submit it when applying for jobs, internships, volunteer positions, or university admission. A well-formatted CV in Word will help you stand out.

CV Formatting Rules
  • Font: Times New Roman or Calibri, size 11–12
  • Margins: Normal (2.54 cm)
  • Length: 1–2 pages for beginners
  • No photos unless specifically requested
  • Save and send as PDF
  • Use clear headings and bullet points

How to Build This CV in Word — Step by Step

Building the CV
  1. 1 Open Word → New Blank Document. Set font to Times New Roman, Size 12
  2. 2 Set margins to Normal. Set line spacing to 1.15
  3. 3 Type your FULL NAME on line 1 → Centre it → Bold → Size 16
  4. 4 Type your contact details (location, phone, email) → Centre → Size 10 → Italic
  5. 5 Add a horizontal line: Insert → Shapes → Line (draws across page)
  6. 6 Type PERSONAL PROFILE → Bold → Size 11 → All Caps → add a bottom border
  7. 7 Write 2–3 sentences about yourself below it. Left-align, size 11
  8. 8 Repeat for EDUCATION, SKILLS, EXPERIENCE, REFEREES
  9. 9 Use bullet points (•) for skills and responsibilities
  10. 10 Use Format Painter to make all section headings look identical
  11. 11 Run Spelling & Grammar check: Review Tab → Spelling & Grammar
  12. 12 Save as Word document: Faith_Otieno_CV.docx
