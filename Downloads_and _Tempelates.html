🔐 Strong Password Generator & Safety Rules

🔑 How to Create a Strong Password

  • ✅ At least 8–12 characters
  • ✅ Uppercase + lowercase letters
  • ✅ Include numbers (0–9)
  • ✅ Include symbols (@, #, $, %, &)
  • ❌ Never use "password", "123456", or your name
  • ❌ Do not use the same password for all accounts

Example: Digi@2026Learn or Cyber#Kenya99

🛡️ Cyber Safety Rules (Poster)

  • ✅ Always log out on public computers
  • ✅ Look for "https://" before entering passwords
  • ✅ Never share your password with anyone
  • ✅ Do not download files from unknown websites
  • ✅ Use antivirus and update regularly
  • ✅ Think before you click – avoid phishing emails

📦 File Size Compression Guide (For Portals)

Problem: e-Citizen and KRA portals reject large files (max 2MB–5MB).

Solution 1 – Compress Images (TinyPNG):
1. Go to tinypng.com
2. Upload image (JPG or PNG)
3. Download compressed version (often 70% smaller)

Solution 2 – Compress PDF (ILovePDF):
1. Go to ilovepdf.com/compress_pdf
2. Upload PDF file
3. Choose compression level (Recommended: "Reduce file size")
4. Download compressed PDF

Solution 3 – Reduce image size manually (Windows):
Right-click image → Open with Paint → Resize → 50% → Save as JPG.

⚡ Quick Troubleshooting (Cybercafé Desk Reference)

❌ "No Internet" – What to do?

  1. Check if Wi-Fi is ON (taskbar icon)
  2. Restart the router (unplug 10 sec)
  3. Forget network and reconnect
  4. Check if other devices have internet
  5. Call ISP if down for >30 min

🔐 "Forgotten Password" – Portal Recovery

  • e-Citizen: Click "Forgot Password" → Email reset
  • KRA iTax: "Forgot Password" → PIN + email
  • Gmail: accounts.google.com → Recovery phone/email
  • HELB portal: Use ID number and registered phone

📎 "File Won't Upload" – Fixes

  • Reduce file size (see compression guide)
  • Change format: JPG instead of PNG, PDF instead of DOCX
  • Rename file – remove spaces and special characters
  • Try a different browser (Chrome, Edge, Firefox)
  • Clear browser cache (Ctrl+Shift+Delete)
