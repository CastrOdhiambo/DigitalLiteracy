CHAPTER TWO

FILE AND FOLDER MANAGEMENT

Introduction
When using a computer, you create many documents such as letters, reports, spreadsheets, pictures, and presentations. These documents are called files. To avoid confusion and loss of work, files must be properly organized. Computers use folders to store and arrange files neatly.

Think of it like this:
A file is like a single paper document. A folder is like a file cover or envelope that holds many papers.

Good file management helps you:
Find your work quickly, Avoid losing important documents, Keep your computer organized, Work more professionally.

In this chapter, you will learn how to create, organize, move, rename, and delete files and folders.

2.1 Understanding Files and Folders

A. What is a File?

A file is a saved piece of work stored on a computer.

Examples of files:
A Word document, An Excel spreadsheet, A PowerPoint presentation, A picture, A PDF document.

Every file has:
I. A File Name, II. A File Type (Extension), III. A Storage Location

Example of a File Name
My_CV.docx
My_CV → File name     .docx → File extension (shows it is a Word document)

Common File Extensions:
1. .docx → Microsoft Word
2. .xlsx → Microsoft Excel
3. .pptx → Microsoft PowerPoint
4. .pdf → PDF Document
5. .jpg / .png → Image file
Understanding file extensions helps you know which program opens the file.

B. What is a Folder?

A folder is a container used to store files. Folders help you group related documents together.

Example:
School Work folder contains: English Essay, Mathematics Assignment, Science Report.
Folders can also contain other folders. These are called subfolders.

C. Opening File Explorer

File Explorer allows you to view and manage files and folders.

Method 1: I. Click the Start Button. II. Click File Explorer.

Method 2: I. Press Windows Key + E on the keyboard. File Explorer will open.

D. Understanding File Explorer Window

When File Explorer opens, you will see:
I. Navigation Pane (Left side - shows folders and drives)
II. Main Window (Displays files and folders)
III. Address Bar (Shows current location)
IV. Search Bar (Used to search for files)

Practical Activity 1

  1. Open File Explorer.
  2. Click on Documents.
  3. Identify at least three files in that location.
  4. Close File Explorer.

Repeat until confident.

Why File Organization Is Important
1. Saves time, 2. Prevents confusion, 3. Makes work professional, 4. Protects important information.

2.2 Creating Files and Folders

Introduction
Creating folders and files is an important skill in digital competency. Before saving any work, you must know where and how to create a proper storage location.

In this section, you will learn how to: Create a new folder, Create a file, Save files in the correct location.

A. Creating a New Folder

Folders help you organize related documents.

Method 1: Creating a Folder on the Desktop
Follow these steps:

  1. Go to the Desktop screen.
  2. Right-click on an empty space.
  3. Select New.
  4. Click Folder.
  5. Type the folder name (e.g., Digital Skills).
  6. Press Enter. The new folder will appear on the desktop.

Method 2: Creating a Folder Inside Documents

  1. Open File Explorer.
  2. Click Documents.
  3. Right-click on an empty space.
  4. Select New.
  5. Click Folder.
  6. Type a name (e.g., School Work).
  7. Press Enter.

B. Creating a File (Example Using MS Word)

To create a file, you must open a program first.

Steps:

  1. Open Microsoft Word.
  2. Click Blank Document.
  3. Type a short paragraph.
  4. Click File.
  5. Select Save As.
  6. Choose the location (e.g., Desktop or Documents).
  7. Type the file name (e.g., Practice Document).
  8. Click Save.

You have now created a file.

C. Important File Naming Rules

When naming files:

  • Use clear and meaningful names
  • Avoid very long names
  • Avoid special symbols such as /\:*?"<>|
  • Use underscores (_) instead of spaces if necessary

Example of good file name:
✓ Term_One_Report     ✗ Document12345

  3. Open Word and type your full name.
  4. Save the file inside the Computer Studies folder.

2.3 Renaming, Copying, Moving and Deleting Files

1. Rename them, 2. Move them to another location, 3. Copy them, 4. Delete them. These are basic but essential digital skills.

A. Renaming a File or Folder

Renaming helps correct mistakes or improve organization.

Steps:

  1. Locate the file or folder.
  2. Right-click on it.
  3. Select Rename.
  4. Type the new name.
  5. Press Enter.

B. Copying a File

Copying creates a duplicate of the file.

Method 1: Using Right-Click
I. Right-click the file. II. Select Copy. III. Open the destination folder. IV. Right-click inside the folder. V. Select Paste. A duplicate file will appear.

Method 2: Using Keyboard Shortcuts
I. Click the file once. II. Press Ctrl + C (Copy). III. Open the destination folder. IV. Press Ctrl + V (Paste).

C. Moving a File

Moving transfers the file to a new location without creating a copy.

Steps:

  1. Right-click the file.
  2. Select Cut.
  3. Open the new location.
  4. Right-click inside the folder.
  5. Select Paste.

OR

  1. Click and hold the file.
  2. Drag it to the new folder.
  3. Release the mouse button.

D. Deleting a File

Deleting removes unwanted files.

Steps:

  1. Click the file once.
  2. Press Delete on the keyboard.

OR

  1. Right-click the file.
  2. Select Delete.

The file moves to the Recycle Bin.

E. Restoring a Deleted File

If you delete a file by mistake:

  1. Open Recycle Bin.
  2. Right-click the file.
  3. Select Restore.

The file will return to its original location.

Practical Activity 3

  1. Create a file called Test File.
  2. Rename it to Final Test File.
  3. Copy it to another folder.
  4. Delete the original file.
  5. Restore it from Recycle Bin.
Common Mistakes to Avoid: Deleting important files accidentally, Forgetting where files are saved.

2.4 Using External Storage Devices (Flash Disk Practice)

Introduction
External storage devices are used to store, transfer, and back up files. The most common external storage device is a USB Flash Disk.

Other examples include: External Hard Drive, Memory Card, External SSD.

In this section, you will learn how to: Insert a flash disk safely, Open it, Copy files to it, Remove it properly.

A. Inserting a Flash Disk

Follow these steps:

  1. Locate a USB port on the computer.
  2. Insert the flash disk carefully into the USB port.
  3. Wait a few seconds for the computer to detect it.
  4. A notification may appear on the screen.

B. Opening the Flash Disk

Method 1: Using Notification
I. Click the notification that appears. II. Select Open Folder to View Files.

Method 2: Using File Explorer
I. Open File Explorer. II. Click This PC. III. Double-click the flash disk drive.

C. Copying Files to a Flash Disk

  1. Locate the file you want to copy.
  2. Right-click the file.
  3. Select Copy.
  4. Open the flash disk.
  5. Right-click inside the flash disk window.
  6. Select Paste.

The file will be copied into the flash disk.

D. Copying Files From a Flash Disk to the Computer

  1. Open the flash disk.
  2. Right-click the file.
  3. Select Copy.
  4. Open the desired folder on the computer.
  5. Right-click and select Paste.

E. Safely Removing a Flash Disk (Very Important)

Never remove a flash disk without safely ejecting it.

Steps:

  1. Locate the USB icon in the taskbar (bottom-right corner).
  2. Click the USB icon.
  3. Select Eject USB Drive.
  4. Wait for the message: "Safe to Remove Hardware."
  5. Remove the flash disk carefully.
Why Safe Removal Is Important: Prevents file corruption, Protects stored data, Avoids damage to the flash disk.

Practical Activity 4

  1. Insert a flash disk.
  2. Create a new folder inside it called Backup Files.
  3. Copy one document into the folder.
  4. Safely eject the flash disk.

2.5 Searching and Organizing Files Efficiently

Introduction
As you continue using a computer, you may store many files. Finding a specific file quickly is an important digital skill. In this section, you will learn how to: Search for files, Sort files, Organize files properly.

A. Searching for a File

Method 1: Using File Explorer Search
I. Open File Explorer. II. Navigate to the location (e.g., Documents). III. Click inside the Search Bar (top-right corner). IV. Type the file name or part of the name. V. Press Enter. The computer will display matching results.

Method 2: Using Start Menu Search
I. Click the Start Button. II. Type the file name. III. Select the file from the results.

B. Sorting Files

Sorting helps arrange files in order.

Steps:

  1. Open a folder.
  2. Right-click inside the folder.
  3. Select Sort by.
  4. Choose one of the following: Name, Date Modified, Type, Size.

C. Organizing Files Properly

Follow these good practices:

  • Create folders for different categories (e.g., School, Work, Personal).
  • Use clear file names.
  • Avoid saving everything on the Desktop.
  • Delete unnecessary files regularly.
  • Back up important documents in a flash disk or cloud storage.

Practical Activity 5

  1. Create three folders named: School, Work, Personal Projects.
  2. Move related files into each folder.
  3. Use the search bar to find one specific file.
  4. Sort the files by date.
Common Mistakes to Avoid: Saving files without checking the location, Using unclear file names, Forgetting to back up important work, Keeping too many files on the desktop.
