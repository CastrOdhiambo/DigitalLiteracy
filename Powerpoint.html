PART IV: PRESENTATIONS (MS POWERPOINT – HANDS ON)

CHAPTER FIVE – INTRODUCTION TO MICROSOFT POWERPOINT

Introduction
Microsoft PowerPoint is used to create visual presentations for:
Lessons and lectures, Business meetings, Projects and reports.
PowerPoint allows you to combine text, images, charts, animations, and transitions to deliver effective presentations.

In this chapter, you will learn how to:
Open PowerPoint and understand its interface, Create and format slides, Insert images, shapes, charts, and text boxes, Apply transitions and animations, Prepare a presentation for slideshow mode and printing.

This chapter is mostly practical.

5.1 Opening PowerPoint and Understanding the Interface

A. Opening Microsoft PowerPoint

There are multiple ways to open PowerPoint:

Method I: Using Start Menu
1. Click the Start Button. 2. Scroll and locate Microsoft PowerPoint. 3. Click to open.

Method II: Using Search
1. Click Start. 2. Type PowerPoint in the search bar. 3. Click Microsoft PowerPoint from the results.

Method III: Desktop Shortcut
1. Locate the PowerPoint icon on the desktop. 2. Double-click it.

When PowerPoint opens, you may see: Blank Presentation → Start from scratch, Templates → Pre-designed slides. Click Blank Presentation to start.

B. Understanding the PowerPoint Interface

The PowerPoint workspace consists of:

  • I. Title Bar – Shows the presentation name and application.
  • II. Quick Access Toolbar – Contains frequently used commands like Save, Undo, Redo.
  • III. Ribbon – Contains Tabs with groups of tools: Home (text formatting, slides), Insert (images, shapes, charts, text boxes), Design (slide themes and backgrounds), Transitions (slide transition effects), Animations (animate objects), Slide Show (present slides), Review (spell check, comments), View (different views of slides).
  • IV. Slides Pane – Displays small previews of all slides on the left. You can select and reorder slides here.
  • V. Slide Workspace – The main area where you edit and design each slide.
  • VI. Notes Pane – Located below the slide workspace. Add speaker notes for your presentation.
  • VII. Status Bar – Shows slide number, theme, and view options.

Practical Activity 1

  1. Open Microsoft PowerPoint.
  2. Identify the Title Bar, Ribbon, Slides Pane, Slide Workspace, Notes Pane, and Status Bar.
  3. Open a Blank Presentation.
  4. In the first slide (Title Slide), type: Title: My First Presentation. Subtitle: Your Name.
  5. Save the presentation as FirstPresentation.pptx.
Tips for Beginners: Always save your presentation after making changes, Explore the Ribbon to familiarize yourself with tools, Start with a blank presentation for practice, Add speaker notes to remember key points during presentation.

5.2 Creating and Formatting Slides

Introduction: Creating and formatting slides allows you to structure your presentation clearly and make it visually appealing.

A. Adding New Slides

Steps: Open your presentation. Go to Home Tab → Slides group → New Slide. Choose a slide layout (Title Slide, Title and Content, Two Content, etc.). The new slide appears in the Slides Pane. Tip: Use layouts that match your content for clarity.

B. Choosing and Changing Slide Layouts

Steps: Select the slide in the Slides Pane. Go to Home Tab → Slides group → Layout. Click a layout (e.g., Title and Content, Comparison, Blank).

C. Formatting Text

Steps: Select the text on the slide. Go to Home Tab → Font group: Change Font Type (e.g., Arial, Calibri), Change Font Size, Apply Bold, Italic, Underline, Change Font Color. Use Paragraph group for alignment: Left, Center, Right, Bulleted or numbered lists.

D. Rearranging Slides

Steps: In the Slides Pane, click the slide you want to move. Drag it to the new position. Release the mouse to place it.

E. Applying Design Themes

Steps: Go to Design Tab. Select a theme from the gallery. Click to apply to all slides. Tip: Themes automatically adjust fonts, colors, and backgrounds.

Practical Activity 1

  1. Open your FirstPresentation.pptx.
  2. Add three new slides using Title and Content layout.
  3. Format each slide: Change the font type to Calibri, Make titles Bold, Center-align main text.
  4. Rearrange slides so the new slides follow the title slide.
  5. Apply a Design Theme of your choice.
  6. Save the presentation.
Tips for Beginners: Use layouts for consistency and clarity, Keep text concise—avoid overcrowding slides, Apply a theme for professional look, Rearrange slides to ensure logical flow.
Common Mistakes to Avoid: Overloading slides with too much text, Using inconsistent fonts and sizes, Forgetting to rearrange slides for proper flow, Ignoring slide layout options.

5.3 Inserting Images, Shapes, and Charts

Introduction: Adding images, shapes, and charts makes your presentation more visual and easier to understand. Visuals help convey information effectively.

A. Inserting Images

Steps: Select the slide where you want the image. Go to Insert Tab → Images group → Pictures. Choose This Device to insert from your computer, or Online Pictures. Browse and select the image → Click Insert. Resize and move the image as needed by dragging corners. Tip: Hold Shift while resizing to keep proportions.

B. Inserting Shapes

Steps: Select the slide. Go to Insert Tab → Illustrations group → Shapes. Click the shape you want (Rectangle, Circle, Arrow, etc.). Click and drag on the slide to draw the shape. Format shape: Fill Color → Change inside color, Outline Color → Change border color, Effects → Add shadow or 3D effect.

C. Inserting Charts

Steps: Select the slide. Go to Insert Tab → Illustrations group → Chart. Choose a chart type (Column, Line, Pie, etc.) → Click OK. A datasheet opens → Enter data. Close the datasheet → The chart updates automatically. Resize or move the chart on the slide.

D. Formatting Visual Elements

Images: Add borders, effects, or crop unwanted parts. Shapes: Adjust size, fill, outline, and effects. Charts: Add titles, data labels, and change colors using the Chart Tools tab.

  4. Insert a Pie Chart showing time spent on different activities (Hobby, School, Sleep, Other).
  5. Format the chart with data labels and colors.
  6. Save the presentation.
Tips for Beginners: Always resize images from corners to maintain proportions, Use shapes to highlight important points, Keep charts simple and readable, Combine visuals with text for clarity.
Common Mistakes to Avoid: Overcrowding slides with too many images or shapes, Stretching images causing distortion, Using too many colors in charts, Forgetting to label charts or shapes.

5.4 Adding Transitions and Animations

Introduction: Transitions and animations make your slides visually appealing and help guide your audience's attention.
- Transition → The effect that occurs when moving from one slide to another
- Animation → The effect applied to objects (text, images, shapes) on a slide

A. Adding Slide Transitions

Steps: Select the slide in the Slides Pane. Go to Transitions Tab. Click a transition effect (e.g., Fade, Push, Wipe). Use Effect Options to change direction or style (if available). Adjust Duration to control speed of transition. Click Apply To All if you want the same transition for all slides. Tip: Keep transitions simple for professional presentations.

B. Adding Animations to Objects

Steps: Select the object on the slide. Go to Animations Tab. Click a desired animation (Entrance, Emphasis, Exit, or Motion Path). Adjust Effect Options to change direction, sequence, or style. Use Animation Pane to view, reorder, or fine-tune animations.

C. Adjusting Timing and Triggers

Steps: Select the animated object. Go to Animations Tab → Timing group. Choose Start: On Click (animates when you click), With Previous (animates with previous animation), After Previous (animates after previous animation finishes). Set Duration → How long the animation lasts. Set Delay → How long before the animation starts.

D. Previewing Transitions and Animations

Go to Slide Show Tab → From Beginning or From Current Slide. Observe transitions between slides and animations on objects. Adjust as needed using the Transitions and Animations Tabs.

Practical Activity 1

  1. Open HobbyPresentation.pptx.

  3. On Slide 2, animate the image of your hobby to Appear with Start: On Click.
  4. Animate the text boxes to Fly In from Left with Start: After Previous.
  5. Use the Animation Pane to reorder and adjust timing.
  6. Preview the slideshow from the current slide.
  7. Save the presentation.
Tips for Beginners: Use subtle transitions for professional presentations, Avoid overusing animations—too many can distract, Preview slides often to check the flow, Adjust timing so animations are smooth and readable.
Common Mistakes to Avoid: Applying too many flashy transitions, Animating every object on every slide, Using inconsistent animation styles, Forgetting to preview and adjust timing.

5.5 Preparing and Delivering a Slide Show

Introduction: After creating slides, transitions, and animations, you need to prepare and deliver your presentation professionally. This includes running the slideshow, printing slides if needed, and sharing your work with others.

A. Slide Show Mode

Steps: Open your presentation. Go to Slide Show Tab → Start Slide Show. Choose: From Beginning → Starts from the first slide, From Current Slide → Starts from the slide you are on. Navigate slides during presentation: Next Slide: Click mouse, press Right Arrow, Down Arrow, or Page Down. Previous Slide: Press Left Arrow, Up Arrow, or Page Up. Go to Slide: Press Number + Enter to jump to a specific slide.

B. Using Presenter View

Presenter View allows you to see notes and upcoming slides while the audience sees only the current slide. Steps: Go to Slide Show Tab → Monitors group → Use Presenter View. Connect to a second monitor or projector. The main screen shows current slide and notes; the audience sees only slides.

C. Printing Slides and Handouts

Steps: Go to File → Print. Choose Printer. Select Print Layout: Full Page Slides → One slide per page, Handouts → Multiple slides per page, Notes Pages → Slide plus speaker notes. Adjust Orientation, Color, and Copies as needed. Click Print.

D. Sharing Presentations

Steps: Go to File → Share. Options include: Email → Send as attachment, Save to Cloud (OneDrive) → Share link, Export → PDF or Video → Convert presentation for easier sharing. Tip: Exporting to PDF ensures formatting remains intact.

Practical Activity 1

  1. Open HobbyPresentation.pptx.

  4. Enable Presenter View and view your notes.
  5. Print the slides as Handouts with 2 slides per page.

Tips for Beginners: Practice navigating slides before the actual presentation, Use Presenter View to remember key points, Print handouts for audience reference, Save in multiple formats for backup and sharing.
Common Mistakes to Avoid: Not rehearsing the slide show, Overloading slides with text and animations, Forgetting to check slide numbers and order, Ignoring file format when sharing.
